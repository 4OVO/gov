# 高频成语800词（中册）

## 使用说明

本成语800词是花生十三老师针对公务员考试总结的高频成语词汇表，涵盖了国考、省考中常见的成语。

- **拼音**：标准汉语拼音标注
- **释义**：精准的成语含义解释
- **考频**：★-★★★★★（星数越多表示考试中出现频率越高）

### 复习建议
1. 优先记忆考频★★★★★和★★★★的成语
2. 注意易混淆成语的区别（如：不孚众望 vs 不负众望）
3. 结合例句理解成语的正确用法
4. 利用碎片化时间定期复习
