# 言语理解
## 一、33555,264111

### 三种文段

1. 说理类文段
   1. 有对策，对策优选。（对策要务实一点）
   2. 无对策，概括问题，有针对问题的对策也可以选。
2. 科普类文段：客观地介绍：
   1. 总分总，分总，重点在总。
   2. 分分，分分分分，重点在全面概括。（A且B,D=B+C）
   3. 中性表达。
3. 新闻类文段
   1. 重点在新。

### 三种关键信息

1. 高频词

2.  主谓宾
3. 首局（主体+主题）（除了背景和背景知识）

### 五种的对策表达方式（说理类文段）

1. 祈使句\建议句：应该、需要、必须、要求、可以、不妨...
2. 禁止型对策：不能、禁止、避免...
3. 逻辑关联词表对策：只有...才...、只要...就...、如果...就...、必须...否则...
4. 必要条件表对策：基础，前提，关键，离不开，必不可少，不可或缺...
5. 语义表对策：时代命题，重要议题，负有...义务，长远目标，必经之路，必由之路，必然趋势，有助于。

### 五种结构

1. 后对策：引出话题+分析论证+对策\观点。
2. 前观点：引出话题/问题/+对策/观点+论证\详细介绍对策。
3. 总分、分总、总分总。
4. 分分：（A且B）、分分分：概括全文。
5. 二层结构
   1. 转折：转折后一般为重点，但有可能转折前后均为重点。
      1. 对比：时间对比：当今为重点。
      2. 对比：国家对比：我国为重点。
   2. 因果\果因：“原因类"选项逆推。
   3. 背景+正文（新）：正文加新。==我国== ==新==
   4. 设问+回答：问题引出+正文对此问题回答。（重点回答）

### 五种话题引出方式

1. 开门见山。
2. 转折对比引出：”前对策“结构经常用转折对比引出话题。
3. 背景\背景知识引出。
4. 由大引小，大集合（所有）引出小集合（有些，特殊）。
5. 设问引出。

### 错误选项俩出处

1. 背景，例子，分析论证中的一般不选。
2. 但是\因此之前的不选、不是...而是...的不是部分、被指代部分一般不选。

### 六有误

1. 杂糅：拼好句。（并列文段除外）
2. 片面：并列文段中常见的错误选项。
3. 绝对化：文段未提及的一般不选。
4. 下定义：说理类文段一般不选；中性表达；一般为科普类文段选项。
5. 本身有误：本身不符合事实；不符合主流价值观；逻辑错误。
6. 无中生有：原文未提及，但存在原文同义替换一定要重点关注。

### 四大类

1. 原因类：
   1. 文段为因果\果因结构。（既成事实）
   2. 文段为原因，选项为结果。
2. 历程类：时间轴。
3. 具体类：1、2、3...
4. 区别比较联系类：双主语。（地位，篇幅相当）

### 一正确

1. 选项为A且B。
2. D=A+C

### 一比较

1. 宽泛VS具体：选具体。
2. 要VS不要：选要。
3. 宏观VS微观：先宏观后微观。

### 一方法：验证方法

1. 根据选项逆推文段进行验证。
2. 揣摩出题老头的意图。

## 二、标题拟定题

1. 属于主旨变形题。
2. 标题中需要包含关键信息。（关键词可适当同义替换，主语正确，体现作者的观点或对策）
3. 有可读性，有趣味性，有通俗易懂，引人入胜的感觉。
4. “ ”、？正确选项概率很高。

## 三、下文推断题

1. 文末新内容——下文解释新内容，尾句起承上启下的作用。
2. 上文提出问题——下文先分析问题，再解决问题。
3. 上文给出宏观对策\原因\问题——下文具体展开。
4. 时间对比，过去不行——下文：现在行。
5. 总分：总，分1，分2，——下文：分3.
6. 转折。
7. 尾句比较——下文一定选择比较的内容。

::: info 注意

不确定选项时候，纠结选项时候，一定要注意文段首句或第二句，但背景和背景知识除外。

:::

## 四、语句填入题

1. 既要把握宏观整体，又要把握微观细节，具体情境具体择优而选。
2. 开头
   1. 前对策，后文为论证。（总分）
   2. 提出问题，尾句肯定是对策或观点。
   3. 总概括：后文分1，分2，分3。
3. 中间
   1. 承上启下
   2. 背景，问题，分析论证，对策\观点。
4. 总结：
   1. 分分总，总是总结句。
   2. 对策句；作者的观点。

## 五、语句排序题

1. 从选项入手，从首句入手，想其他。
2. 宏观把握：
   1. 结构
      1. 说理类：前对策，后对策。
      2. 科普类：分1，分2，分3。
      3. 新闻类：背景+新。
   2. 分堆
   3. 定义
   4. 引用
   5. 固定顺序（时间轴）
3. 微观把握
   1. 首句：下定义、由大引小、时间、背景。
   2. 尾句：总结、展望未来。
   3. 首位呼应：
   4. 捆绑\组合：指代词，相同词。（可以在捆绑之中找顺序）

::: info 注意

找顺序，不应先考虑捆绑！！！

:::

## 六、细节判断题

1. 偷换概念。
2. 混淆时态。
3. 杂糅。
4. 本身有误。（不符合实际）
5. 绝对化。
6. 无中生有。
