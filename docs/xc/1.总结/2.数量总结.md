## 工程问题

::: info 注意

1. 工作总量=时间×效率
2. 时间和效率成反比。
3. 工作时间和时间或效率成正比。

:::

1. 给出各个工作段工作的时间：设工作总量的时间为最小公倍数。
2. 给出各个工作段工作的效率：将效率之间的比值当作效率。

## 数字特性

### 整除

1. $y=\frac{a}{b}x$，注意：$\frac{a}{b}$是最简分数，x为b的倍数，y为a的倍数。
1. a，b都是x的倍数。a$\pm$b也为x的倍数。
3. 倍数特性：
   1. 3：各个数字之和，消除本身是3的倍数，然后求和。
   2. 2，5：看末尾。
   3. 4，25：看后两位。
   4. 8，125：看后三位。


### 余数

1. 余同取余。eg：/4...1，/5...1,取余为20n+1。==注意：0==
2. 和同加和。eg：/4...2，/5...1,取和为20n+6。
3. 差同减差。eg：/6...4，/7...5,取和为42n-2。

### 等差数列

1. $a_n=a_1+(n-1)d$
2. $a_n=a_m+(n-m)d$
3. $S_n=\frac{a_1+a_n}{2}=n*a_\frac{n+1}{2}$
4. 中项=平均数=$\frac{a_1+a_n}{2}$

## 经济利润问题

::: info 注意 

经济利润问题和资料的ABRX一致。

A:成本

B:售价

R:利润率

X:利润

:::

1. 基本公式
   1. 售价=成本+利润=成本×(1+利润率)
   2. $\text{成本}=\frac{\text{售价}}{\text{1+利润率}}$
   3. ==数量：==$\text{利润率}=\frac{\text{利润}}{\text{成本}}$、==资料：==$\text{利润率}=\frac{\text{利润}}{\text{收入}}$
   4. 折扣n=售价×0.n

2. 阶梯计费
   1. 分段计算。

3. 分批销售
   1. 每批收入之和=总收入=成本+利润

4. 函数最值
   1. y=(30-3x)(x+4)，令y=0，则$x_1$=10，$x_2$=-4，最值为$\frac{10+(-4)}{2}$=3。
   2. y=$ax^2$+bx+c(a$\not=$0)，则最值为  - $\frac{b}{2a}$。

5. 增长率
   1. 间隔增长率：$R=R_1+R_1+R_1R_2$：求：==涨价、降价==
   2. 比值增长率：（$A=\frac{B}{C}$）：$R=\frac{R_1-R_2}{1+R_2}$：求：==$\text{单价}=\frac{\text{收入}}{\text{销量}}$==
   3. 乘积增长率：A=B×C：$R=R_1+R_1+R_1R_2$：求：==收入=单价×销量==
   4. 盐水：A=B+C：售价=成本+利润


## 几何公式

### 面积体积

1. 周长：l=$\frac{n \pi r}{180}$+2r
2. 面积
   1. 正三角形：$S=\frac{\sqrt{3}}{4}a^2$
   2. 菱形：S=$\frac{1}{2}$(长对角线×短对角线)
   3. 梯形：S=$\frac{1}{2}$(上底+下底)×高
   4. 扇形：S=$\frac{1}{2}rl$=$\frac{n \pi r^2}{360}$
   5. 球体：S=4$\pi$R^2
3. 体积
   1. 锥体：V=$\frac{1}{3}$底×高
   2. 球体：V=$\frac{4}{3} \pi R^3$

### 勾股定理

1. $a^2=b^2+c^2$
   1. 3\4\5
   2. 5\12\13
   3. 6\8\10
   4. 7\24\25
   5. 8\15\17
   6. 9\40\41
2. 30°、60°、90°：1、2、$\sqrt{3}$
3. 45°、90°：1、1、$\sqrt{2}$

### 三角形泛型

1. 边(a)：面积($a^2$)：体积($a^3$)
2. 三角形同底等高，面积相等。
3. 任意四边形，对角线分成的四个块，对角面积乘积相等。
4. 任意梯形，对角线分成的四个块，相邻面积之比==等于==一条对角线左右两部分之比==等于==上底下底之比。
5. 任意梯形，对角线分成的四个块，对角面积之比=$(\frac{\text{上底}}{\text{下底}})^2$

### 几何最值

1. 矩形周长一定时，正方形面积最大。
2. 矩形面积一定时，正方形周长最小。
3. 周长一定时，越趋于圆，面积越大。
4. 表面积一定时，越趋于球，表面积越大。

### 最短路径

1. 过河路径最短：过A做河的垂线，距离相等的点B,链接BC交河于点O，AO+OC即为最短路径。
2. 两点在几何体的不同面，展开连接直线，路径最短。

## 行程问题

1. 基础行程问题
   1. 路程=速度×时间
2. 比例法
   1. 路程一定，速度和时间成反比。
   2. 速度一定，路程和时间成正比。
   3. 时间一定，路程和速度成正比。
3. 相遇追及问题
   1. 直线追及：路程差=速度差×时间。
   2. 直线相遇：路程和=速度和×时间。
   3. 环形追及：追上n次，多走n圈。$nS=(v_1-v_2)t$
   4. 环形相遇：相遇n次，累计走n圈。$nS=(v_1+v_2)t$
4. 流水问题
   1. $v_\text{顺}=v_\text{船}+v_\text{水}$
   2. $v_\text{逆}=v_\text{船}-v_\text{水}$
   3. $v\text{船}=\frac{v_\text{顺}+v_\text{逆}}{2}$
   4. $v\text{水}=\frac{v_\text{顺}-v_\text{逆}}{2}$
5. 火车过桥问题
   1. S=桥长+车长
6. 平均速度问题
   1. $v=\frac{s}{t}$
   2. 等时间：$v=\frac{v_1+v_2}{2}$
   3. 等距离：$v=\frac{2v_1v_2}{v_1+v_2}$
7. 匀加速问题
   1. $v=\frac{v_\text{初}+v\text{末}}{2}$
   2. $v_\text{末}=v_\text{初}+at$
8. 多次相遇问题
   1. 相遇n次，累计路程和为==2n-1==个全程。
   2. 相遇n次，每次相遇的时间为==2n-1×第一次相遇的时间==。

## 趣味杂题

### 牛吃草问题

1. $\text{白吃牛}=\frac{\text{天}  _ 1\text{牛} _1-\text{天} _ 2 \text{天} _ 2}{\text{天} _1-\text{天} _ 2}$
2. 原有草=（牛-白吃牛）× 天数
3. 给时间，问牛。 牛=$\frac{\text{原有草}}{\text{时间}}$+白吃牛
4. 给牛，问时间。时间=$\frac{\text{原有草}}{\text{牛-白吃牛}}$

### 鸡兔同笼问题

1. 较小的数值=$\frac{\text{理论-实际}}{\text{插值}}$

### 方阵问题

1. n行n列，总人数$n^2$。最外层4(n-1)，相邻两层差8

### 植树问题

1. 两端不植树=间隔数-1。
2. 两端不植树=间隔数+1。
3. 一端植树，一端不植树。或环形植树=间隔数。

### 统筹规划问题

1. 工程统筹：各自做自己擅长的部分。
2. 经济统筹：优先选单价低的，尽量不浪费。
3. 天平问题：$3^n$=总数，n次找到。

### 比赛问题

1. 单循环：单个队伍。$C_n^2$
2. 双循环：$2C_n^2$
3. 淘汰赛：
   1. $2^n$=总的队伍数，需要打n场比赛。
   2. 场次：2n个队伍，有n场比赛。若队伍数为奇数，则轮空一个队伍。
