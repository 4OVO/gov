### A

1. 一般基期：假设分配，415 法，A=$\frac{X}{R}$
2. 间隔基期：1.先求间隔增长率=R1+R2+R1R2，2.再按一般基期求。
3. 基期核查
4. 已知变化情况求基期，==A=$\frac{X}{R}$注意：A 是基期==
   ::: info 例题：已知每公顷增产 63.3kg，提高约 6%，则 A=$\frac{63.3}{6}$=1055kg/公顷
   :::

### B

1. 假设 X 求后期，列不等式方程
2. 假设 R 求后期，一年一年推 B=A+AR=A(1+R),或者 R=R1+R2+R1R2
3. 按照实际 R 求后期，B=B×$\frac{1+R~\text{实际}~}{1+R~\text{名义}~}$

### R

1. 一般增量：假设分配，415 法（X=AR,X=$\frac{B}{1+R}$R，注意 R 极小时 X 可直接用 BR 省略计算）
2. 增量倍数或比值。$\frac{X1}{X2}$
3. 整体增量 X/部分增量 Xn，X=X1+X2+X3+...+Xn

### X

1. 一般增速：R=$\frac{X}{A}$=$\frac{B-A}{A}$=$\frac{B}{A}$-1
2. 间隔增长率 R=R1+R2+R1R2, (10%X10%=1%),注意逆用，注意负数。
3. 比值增长率 R=$\frac{1+R1}{1+R2}$-1=$\frac{R1-R2}{1+R2}$ ------A=$\frac{B}{C}$
4. 乘积增长率 R=R1+R2+R1R2 ------A=BxC
5. 增长率=增长倍数-1，增长倍数=增长率+1

## 比重

### 单期比重

1. 一般比重类：比重=$\frac{\text{部分}}{\text{整体}}$------整体=$\frac{\text{部分}}{\text{占比}}$------部分=整体 x 占比
2. 部分比重和/差：比重=$\frac{\text{部分和/差}}{\text{整体}}$
3. 饼图
4. 基期比重：基期比重=现期比重 x$\frac{1+R~\text{整体}~}{1+R~\text{部分}~}$ ==注意：现期比重题目有时会给，需要特别注意一下==
5. 隔级比重：
   1. $\frac{\text{小}}{\text{大}}$=$\frac{\text{小}}{\text{中}}$x$\frac{\text{中}}{\text{大}}$
   2. $$
                  \frac{\text{小}}{\text{中}}=\frac{\frac{\text{小}}{\text{大}}}{\frac{\text{中}}{\text{大}}}
      $$

::: info 例题

医院占全国医疗卫生机构的 77.1%，公立医院站医院的 75.7%，则：公立医院占全国医疗卫生机构的？

$$
\frac{\text{公立医院}}{\text{全国医疗卫生机构}}=\frac{\text{公立医院}}{\text{医院}}×\frac{\text{医院}}{\text{全国医疗卫生机构}}=75.7\%×77.1\%=58.4\%
$$

:::

### 两期比重

1. 比重趋势：

   1. R1>R2,比重上升
   2. R1<R2,比重下降

2. 比重趋势进一步

::: info 例题

某一年，东南西北分别占全国的 76.6%，12%，9.4%，2.0%，同比增长 19.8%，17.1%，17.6%，10.2%，全国同比增长 19.1%，则，上一年的比重是？

19.8>19.1,比重上升，则东比重小于 76.6

12<17.1,则比重下降，则西比重大于 12

:::

3. 逆应用

   1. 比重上升，R1>R2
   2. 比重下降，R1<R2

4. 比重差：比重差=现期比重-基期比重=$\frac{\text{基期部分}}{\text{现期整体}}$×(部分 R-整体 R)
   1. 比重差的绝对值<增速差的绝对值
   2. 注意部分 R-整体 R 是负数
   3. ==注意：比重求不了增速==

## 比较

### 常见方法

1. 趋势法

   1. 分子增速大，则分数大。
   2. 分母增速大，则分数小。

2. 通分

   1. 分子相同，分母小则分数大。
   2. 分母相同，分子大则分数大。

3. 拆分

   1. 同王者荣耀法

   ::: info 例题

   北方学生 2701 人，全国 5412 人，北方占全国是否超过$\frac{1}{2}$

   $\frac{2701}{5412}$=$\frac{2700+1}{5400+12}$,$\frac{1}{12}$<$\frac{1}{2}$,则没超过

   :::

4. 双线法

### 替代比较

1. 年均增长率：$\frac{\text{末期}}{\text{基期}}$=(1+R)^n^~~~$\frac{\text{末期}}{\text{基期}}$
2. 倍数=增长率+1,R=$\frac{X}{A}$=$\frac{B-A}{A}$=$\frac{B}{A}$-1~~~$\frac{X}{A}$
3. $\frac{A}{A+B}$~~~$\frac{A}{B}$
4. R=$\frac{X}{A}$~~~$\frac{X}{B}$

### 比值

A=$\frac{B}{C}$：方法同上

### 增量大小比较

1. X=AR=$\frac{B}{1+R}$×R
2. B 越大 R 越大，则 X 越大。
3. 一大一小看倍数。（解释：我的 B 是你的 n 倍，你的 R 是我的 n 倍+，我们的 X 才可能相等）
