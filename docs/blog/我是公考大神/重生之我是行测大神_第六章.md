# 第六章 资料分析速算

周二下午三点，林默准时来到图书馆，苏梦已经在那里等他了。她穿着一件浅绿色的上衣，看起来清新可人。

"今天我们重点学习资料分析的速算技巧，"林默说道，"资料分析是行测中最容易提分的部分，也是最需要速算技巧的部分。"

苏梦点了点头："好，我资料分析总是做得很慢，希望能通过今天的学习提高速度。"

林默翻开资料分析教材，找到一道例题："你看这道题，2017年全国固定资产投资（不含农户）631684亿元，比上年增长7.2%，增速比上年提高0.9个百分点。其中，民间固定资产投资381510亿元，增长6.0%，增速比上年提高2.8个百分点。问2016年全国固定资产投资（不含农户）是多少？"

苏梦看了看题目，想了想："应该用基期量=现期量/(1+增长率)来算，631684除以1.072，等于多少呢？"

她拿出计算器准备计算，林默却拦住了她："不用计算器，用花生十三的截位直除法。把1.072近似看作1.07，631684除以1.07，等于多少？"

苏梦想了想："631684除以1.07，大概是589000左右。"

"对，"林默说道，"正确答案是589258亿元，和你的估算差不多。这就是截位直除法的优势，不用精确计算，通过估算就能快速得出答案。"

接下来，林默给苏梦讲解了花生十三的资料分析速算技巧，包括截位直除法、估算法、特殊分数法、化同法、差分法等。他讲得非常详细，每一种技巧都举了几个例子，苏梦听得很认真，不时点头表示理解。

"林默，你看这道题，"苏梦指着一道题说道，"2017年全国房地产开发投资109799亿元，比上年增长7.0%，增速比上年提高0.1个百分点。其中，住宅投资75148亿元，增长9.4%，增速比上年提高2.5个百分点。问2017年住宅投资占全国房地产开发投资的比重是多少？"

"这道题应该用比重=部分/整体来算，"林默说道，"75148除以109799，等于多少？"

苏梦想了想："75148除以109799，大概是68.4%左右。"

"对，"林默说道，"正确答案是68.4%。你是怎么算的？"

"我用了截位直除法，"苏梦说道，"把109799近似看作110000，75148除以110000，等于0.683，也就是68.3%，和正确答案差不多。"

"很好，"林默说道，"你已经掌握了截位直除法的精髓。"

两人一直学到中午，然后一起去食堂吃饭。吃饭时，苏梦问道："林默，你说资料分析的速算技巧需要多长时间才能掌握？"

"因人而异，"林默说道，"一般来说，只要每天练习20道资料分析题，一个月就能掌握这些技巧。"

"那我每天都练习20道资料分析题，"苏梦说道，"争取一个月内掌握这些技巧。"

林默点了点头："好，这样很好。"

吃完饭，两人回到图书馆，继续学习。林默给苏梦讲解了花生十三的资料分析常考公式，包括基期量、现期量、增长率、增长量、比重、平均数、倍数等。苏梦学得很认真，很快就掌握了这些公式。

"林默，你看这道题，"苏梦指着一道题说道，"2017年全国社会消费品零售总额366262亿元，比上年增长10.2%。问2017年全国社会消费品零售总额比上年增长了多少亿元？"

"这道题应该用增长量=现期量×增长率/(1+增长率)来算，"林默说道，"366262×10.2%/(1+10.2%)，等于多少？"

苏梦想了想："10.2%近似看作1/10，所以366262×(1/10)/(1+1/10)=366262/11≈33296亿元。"

"对，"林默说道，"正确答案是33296亿元。你用了特殊分数法，这是花生十三最常用的速算技巧之一。"

"是的，"苏梦说道，"特殊分数法真的很有用，能大大提高计算速度。"

林默笑着说道："那当然，花生十三的技巧都是经过实战检验的，特别适合考试。"

两人一直学到晚上八点，然后一起走出图书馆。苏梦说道："林默，今天的学习收获很大，我觉得我的资料分析速度提高了很多。"

林默笑了笑："只要你坚持练习，你的资料分析速度会越来越快，正确率会越来越高。"

"谢谢你，林默，"苏梦说道，"如果没有你，我可能永远都不会掌握这些技巧。"

"不用谢，"林默说道，"我们是朋友，应该互相帮助。"

苏梦笑着点了点头，转身离开了。林默看着她的背影，心中涌起一股暖流。他知道，这一次，他一定能和苏梦在一起，创造属于他们的幸福生活。

回到宿舍，林默躺在床上，回想着今天和苏梦的相处，觉得很开心。他拿出笔记本，开始总结今天的学习内容，然后制定明天的学习计划。

突然，他的手机响了，是中公教育的王老师打来的。

"你好，林默同学，"王老师说道，"我们听说你行测学得很好，特别是资料分析，我们想邀请你参加我们的公考培训课程，担任助教。"

林默愣了一下，然后说道："谢谢邀请，但是我现在正在备考公务员，没有时间担任助教。"

"没关系，"王老师说道，"我们可以等你考上公务员之后再邀请你。对了，我们最近有一个资料分析的专题讲座，想请你去做嘉宾，分享一下你的学习经验，你有兴趣吗？"

林默想了想，说道："不了，谢谢，我现在有自己的学习计划，就不参加了。"

"好吧，"王老师说道，"如果你有兴趣的话，随时可以联系我们。"

挂断电话，林默心中感慨道，看来他的资料分析水平已经引起了培训机构的注意。不过，他现在的首要目标是考上公务员，所以暂时不想考虑其他事情。

林默躺在床上，想着苏梦，想着自己的未来，觉得充满了希望。他知道，有了重生的机会和花生十三的技巧，他一定能实现自己的目标，创造属于自己的传奇。
