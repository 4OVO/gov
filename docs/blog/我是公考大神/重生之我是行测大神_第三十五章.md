# 第三十五章 成绩公布

国考成绩公布的日子终于到了。林默和苏梦早早地起床，洗漱完毕，坐在电脑前，等待国家公务员局网站开通成绩查询通道。

**等待查询**

早上八点，国家公务员局网站公布了国考成绩查询通知。林默和苏梦立即登录网站，输入身份证号和准考证号，点击查询按钮。此时，他们的心跳加速，手微微发抖，眼睛紧盯着电脑屏幕，期待着成绩的出现。

**成绩揭晓**

电脑屏幕上显示出成绩的那一刻，林默和苏梦都惊呆了。林默的行测成绩是92分，申论成绩是78分，总分170分；苏梦的行测成绩是85分，申论成绩是72分，总分157分。两人的成绩都远远超过了报考职位的分数线，顺利进入了面试。

**惊喜与激动**

“林默，我考了157分！”苏梦激动地喊道，眼睛里闪烁着泪光。“我考了170分！”林默也兴奋地喊道，一把将苏梦抱在怀里。两人紧紧相拥，庆祝这来之不易的成功。

**分享喜悦**

林默和苏梦第一时间将成绩分享给了学习小组的其他同学。张阳、李明、王芳等人收到消息后，纷纷发来祝贺的信息。张阳说：“恭喜你们！林默170分，苏梦157分，真是太厉害了！”李明说：“我就知道你们能考上，花生十三的技巧果然名不虚传！”王芳说：“太棒了，你们是我们学习的榜样！”

**回顾备考**

林默和苏梦开始回顾这段备考的日子。林默说：“现在回想起来，重生真是一件神奇的事情。如果不是重生，我可能永远不会知道花生十三的技巧，也不会考上公务员。”苏梦说：“是的，感谢重生，让我们相遇，让我们一起努力，一起成功。”

**感谢花生十三**

林默和苏梦都非常感谢花生十三的行测和申论技巧。林默说：“花生十三的技巧真的很实用，尤其是资料分析的速算技巧，帮我节省了很多时间，提高了正确率。”苏梦说：“是的，申论的关键词提取和结构安排技巧，让我的申论成绩提高了很多。”

**面试准备**

林默和苏梦知道，进入面试只是第一步，面试同样重要。他们开始更加认真地准备面试，购买了更多的面试辅导资料，参加了面试培训班，互相模拟面试，提高面试能力。

**公益基金启动**

林默和苏梦决定正式启动公考公益基金。他们利用微博和微信公众号的广告收入，首批资助了10名家庭困难的公考考生，每人资助3000元，帮助他们购买备考资料和参加培训班。这一举措得到了很多人的赞赏和支持，粉丝数量进一步增长。

**未来展望**

林默和苏梦对未来充满了希望。他们计划考上公务员后，继续经营微博和微信公众号，分享学习方法和技巧，帮助更多的人考上公务员。他们还计划扩大公考公益基金的规模，资助更多的家庭困难考生。同时，他们也开始规划未来的生活，希望能早日结婚，一起过上幸福的生活。

**结语**

国考成绩的公布，标志着林默和苏梦备考阶段的成功结束，也标志着他们人生新的开始。他们知道，未来还有更多的挑战和机遇在等待着他们，但他们相信，只要一起努力，相互支持，就一定能克服所有困难，实现自己的梦想。

林默和苏梦的故事告诉我们，努力是成功的必要条件，但正确的方法和技巧同样重要。花生十三的行测和申论技巧，帮助他们在国考中取得了优异的成绩，实现了公务员梦想。同时，他们的故事也告诉我们，爱情和友情的力量是无穷的，相互支持、共同努力，才能走得更远。

愿所有公考考生都能找到适合自己的学习方法和技巧，实现自己的公务员梦想！
