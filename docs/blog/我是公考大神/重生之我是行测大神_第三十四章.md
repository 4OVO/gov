# 第三十四章 等待成绩

国考结束后，林默和苏梦进入了漫长的等待成绩阶段。这段时间里，他们的心情既期待又焦虑，每天都在关注国家公务员局的网站，希望能早日查到成绩。

**日常安排**

为了缓解等待成绩的焦虑，林默和苏梦制定了详细的日常安排：上午复习面试知识，下午进行体育锻炼，晚上在微博和微信公众号上分享学习方法和技巧。这样的安排让他们的生活充实而有意义，也减少了对成绩的过度关注。

**面试准备**

林默和苏梦知道，即使笔试成绩很高，面试也不能掉以轻心。他们购买了面试辅导资料，学习面试的基本流程、题型和技巧。他们还会互相模拟面试，一个扮演考官，一个扮演考生，通过模拟练习提高面试能力。

**微博和微信公众号**

林默和苏梦继续在微博和微信公众号上分享学习方法和技巧，帮助更多的人提高行测和申论成绩。他们发布了详细的国考真题解析、面试技巧指导等内容，得到了很多人的关注和点赞。粉丝数量从最初的几千人增长到了几万人，成为了公考领域的知名账号。

**学习小组活动**

学习小组的其他同学也在等待成绩，他们会定期聚会，交流等待成绩的心情和面试准备的情况。张阳说：“虽然等待成绩很焦虑，但想到我们已经尽力了，就觉得不管结果如何，都不会后悔。”李明说：“是的，我们已经掌握了花生十三的技巧，即使这次没考上，下次也一定能考上。”王芳说：“我现在每天都在看面试资料，为面试做准备，希望能一次成功。”

**公益基金准备**

林默和苏梦开始着手准备公考公益基金的事情。他们制定了详细的计划，包括基金的资金来源（主要是微博和微信公众号的广告收入）、资助对象（家庭困难的公考考生）、资助标准（每人1000-5000元，根据困难程度确定）等。他们还联系了学校的相关部门，希望能得到学校的支持和指导。

**心情变化**

随着成绩公布日期的临近，林默和苏梦的心情变得越来越焦虑。苏梦说：“林默，我最近总是梦见国考成绩公布，有时候梦见考得很好，有时候梦见考得不好，醒来后心情就会很复杂。”林默说：“我也是，不过我觉得我们已经尽力了，不管结果如何，我们都应该接受。”

**成绩公布前的晚上**

成绩公布前的晚上，林默和苏梦一起在图书馆学习。他们虽然在看书，但心思却不在书上，一直在想着明天的成绩。苏梦说：“林默，明天成绩就要公布了，你紧张吗？”林默说：“有点紧张，但更多的是期待。不管结果如何，我们都已经努力了，这就足够了。”苏梦说：“是的，我们一起努力，一起进步，这段经历已经是我们宝贵的财富了。”

**睡前的鼓励**

回到宿舍后，林默给苏梦发了一条微信：“不管明天成绩如何，你都是最棒的，我永远支持你。”苏梦回复：“你也是，不管结果如何，我们都要一起面对，一起努力。晚安，明天加油！”林默回复：“晚安，明天一起查成绩。”

**等待的意义**

躺在床上，林默想起了这段等待成绩的日子。他觉得，等待也是一种成长，它让他学会了耐心，学会了接受不确定性，学会了珍惜当下。他知道，不管明天的成绩如何，他都会继续努力，继续前进，因为他有苏梦在身边，有朋友们的支持，有对未来的信心。

苏梦也躺在床上，想着明天的成绩。她知道，成绩只是一个结果，重要的是过程。在这段备考的日子里，她不仅掌握了知识和技巧，还收获了爱情和友情，这些都是比成绩更重要的财富。她相信，不管明天的成绩如何，她的未来都会很美好，因为她有林默在身边，有对生活的热爱，有对梦想的追求。
