# 第十四章 培训机构注意

周三上午，林默和苏梦一起去图书馆学习。他们刚坐下，就看到中公教育的王老师走了过来。

"林默，苏梦，你们好，"王老师说道，"我们中公教育想邀请你们参加我们的公考培训课程，担任助教。"

"王老师，我们现在正在备考公务员，没有时间担任助教，"林默说道。

"没关系，"王老师说道，"我们可以等你们考上公务员之后再邀请你们。对了，我们中公教育想和你们合作，把你们的行测和申论技巧整理成课程，分享给更多的同学。"

"合作？"林默问道，"怎么合作？"

"我们中公教育可以提供平台和资源，你们提供技巧和经验，我们共同开发一门公考培训课程，"王老师说道，"课程收入我们可以分成。"

林默想了想："我们现在正在备考公务员，没有时间开发课程。等我们考上公务员之后，再考虑合作的事情吧。"

"好的，"王老师说道，"我们等你们考上公务员之后，再谈合作的事情。"

王老师离开了，林默和苏梦继续学习。他们觉得培训机构的邀请是对他们的认可，但是现在他们的首要目标是考上公务员，所以暂时不想考虑其他事情。

中午，林默和苏梦一起去食堂吃饭。吃饭时，苏梦问道："林默，你觉得我们应该和培训机构合作吗？"

"等我们考上公务员之后，再考虑合作的事情，"林默说道，"现在我们的首要目标是考上公务员，其他事情都可以暂时放一放。"

"好，"苏梦说道，"我们先考上公务员，然后再考虑其他事情。"

吃完饭，两人回到图书馆，继续学习。他们刚坐下，就看到华图教育的李老师走了过来。

"林默，苏梦，你们好，"李老师说道，"我们华图教育想邀请你们参加我们的公考培训课程，担任助教。"

"李老师，我们现在正在备考公务员，没有时间担任助教，"林默说道。

"没关系，"李老师说道，"我们可以等你们考上公务员之后再邀请你们。对了，我们华图教育想和你们合作，把你们的行测和申论技巧整理成课程，分享给更多的同学。"

"我们现在正在备考公务员，没有时间开发课程，"林默说道，"等我们考上公务员之后，再考虑合作的事情吧。"

"好的，"李老师说道，"我们等你们考上公务员之后，再谈合作的事情。"

李老师离开了，林默和苏梦继续学习。他们觉得越来越多的培训机构注意到了他们，这是对他们的认可，但是现在他们的首要目标是考上公务员，所以暂时不想考虑其他事情。

下午，林默和苏梦在图书馆学习时，遇到了粉笔教育的张老师。张老师也想邀请他们参加粉笔教育的公考培训课程，担任助教，并且想和他们合作开发课程。林默和苏梦同样以备考公务员为由，拒绝了张老师的邀请，说等考上公务员之后再考虑合作的事情。

离开图书馆时，苏梦说道："林默，今天有三个培训机构的老师来邀请我们，看来我们真的很受关注。"

"是的，"林默说道，"这说明我们的行测和申论技巧很有用，得到了培训机构的认可。"

"那我们考上公务员之后，应该和培训机构合作吗？"苏梦问道。

"可以考虑，"林默说道，"我们可以把花生十三的技巧分享给更多的同学，帮助他们考上公务员。"

"好，"苏梦说道，"我们考上公务员之后，就和培训机构合作，分享花生十三的技巧。"

回到宿舍，林默躺在床上，回想着今天培训机构的邀请。他觉得这是对他的认可，但是现在他的首要目标是考上公务员，所以暂时不想考虑其他事情。他决定，等考上公务员之后，再考虑和培训机构合作的事情。

林默看着自己的未来规划，心中充满了希望。他知道，有了重生的机会和花生十三的技巧，他一定能实现自己的目标，创造属于自己的传奇。
