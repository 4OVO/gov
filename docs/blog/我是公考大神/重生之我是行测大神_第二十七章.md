# 第二十七章 冲刺计划

周二上午，林默和苏梦一起去图书馆学习。他们已经完成了国考报名，现在是时候制定详细的冲刺计划，争取在国考前进一步提高成绩。

"林默，我们已经完成了国考报名，现在应该制定一个详细的冲刺计划，"苏梦说道，"你觉得我们的冲刺计划应该包括哪些内容？"

"我们的冲刺计划应该包括以下内容，"林默说道，"每天学习行测和申论各四个小时，每周参加一次模考，每天总结错题，每天积累常识和申论素材，同时在微博和微信公众号上分享学习方法和技巧，帮助更多的人提高成绩。"

"好，"苏梦说道，"我们就按照这个冲刺计划执行，争取在国考前进一步提高成绩。"

林默和苏梦开始制定详细的冲刺计划，包括每天的学习时间、学习内容、模考安排等。他们的冲刺计划如下：

**冲刺计划**

**一、每天学习时间**
- 上午：8:00-12:00，学习行测
- 下午：14:00-18:00，学习申论
- 晚上：19:00-21:00，总结错题和积累素材

**二、每周学习内容**
- 周一：资料分析和归纳概括
- 周二：言语理解和综合分析
- 周三：判断推理和提出对策
- 周四：数量关系和贯彻执行
- 周五：常识判断和申发论述
- 周六：模考和错题总结
- 周日：学习小组活动和技巧分享

**三、模考安排**
- 每周六上午：参加中公教育的模考
- 每周六下午：总结模考错题和分析原因

**四、错题总结**
- 每天晚上：总结当天的错题，分析错误原因
- 每周日：整理一周的错题，形成错题本

**五、常识和申论素材积累**
- 每天早上：阅读人民日报和新华网的时政新闻
- 每天晚上：积累常识知识点和申论素材

**六、网络分享**
- 每天晚上：在微博和微信公众号上分享学习方法和技巧
- 每周日：发布一篇详细的学习经验和技巧总结

制定完冲刺计划后，林默和苏梦开始按照计划执行。他们的学习效率比以前更高了，因为他们现在有了详细的计划，知道每天应该学习什么内容，应该如何安排时间。

中午，林默和苏梦一起去食堂吃饭。吃饭时，苏梦说道："林默，我们的冲刺计划很详细，你觉得我们能按照计划执行吗？"

"能，"林默说道，"我们已经制定了详细的计划，只要有信心和毅力，一定能按照计划执行。"

"我相信你，"苏梦说道，"有你在身边支持我，我觉得很有信心。"

"我也相信你，"林默说道，"你是最棒的。"

吃完饭，两人回到图书馆，继续学习。林默给苏梦讲解了花生十三的资料分析速算技巧，苏梦听得很认真，很快就掌握了这些技巧。他们的学习效率比以前更高了，因为他们现在不仅是学习伙伴，还是男女朋友，互相鼓励，互相支持。

下午，林默和苏梦在图书馆学习时，遇到了他们的辅导员李老师。李老师看到他们在学习，就走了过来。

"林默，苏梦，你们好，"李老师说道，"我听说你们已经制定了详细的冲刺计划，是吗？"

"是的，李老师，"林默说道，"我们已经制定了详细的冲刺计划，包括每天的学习时间、学习内容、模考安排等。"

"很好，"李老师说道，"制定详细的计划是备考的重要环节，有了计划，学习就有了方向和动力。我相信你们一定能按照计划执行，在国考中取得好成绩。"

"谢谢李老师的鼓励，"林默说道，"我们会继续努力的。"

李老师离开了，林默和苏梦继续学习。他们觉得辅导员的鼓励是对他们的认可，更加坚定了他们按照计划执行的信心。

晚上，林默和苏梦一起去食堂吃饭。吃饭时，苏梦说道："林默，我们的冲刺计划很详细，但是执行起来可能会很辛苦，你觉得我们能坚持下去吗？"

"能，"林默说道，"只要我们有信心和毅力，一定能坚持下去。而且，我们现在是男女朋友，互相鼓励，互相支持，一定能度过这段艰苦的备考期。"

"我也是这么想的，"苏梦说道，"有你在身边，我觉得再苦再累也值得。"

吃完饭，两人回到图书馆，继续学习。他们总结了当天的错题，分析了错误原因，然后在微博和微信公众号上分享了学习方法和技巧。他们的粉丝数量不断增加，越来越多的人开始关注他们，向他们请教学习方法和技巧。他们耐心地回复每一条私信和评论，分享花生十三的行测和申论技巧，帮助更多的人提高成绩。

离开图书馆时，苏梦说道："林默，今天的学习很充实，我觉得我们的冲刺计划很有效果。"

"是的，"林默说道，"只要我们按照计划认真执行，一定能在国考前进一步提高成绩，实现我们的目标。"

"我们的目标是什么？"苏梦问道。

"我们的目标是考上公务员，然后结婚，一起过上幸福的生活，"林默说道，"同时，我们要帮助更多的人考上公务员，让他们也能实现自己的梦想。"

"是的，"苏梦说道，"这是我们共同的目标。"

两人手牵着手，一起走回了宿舍。他们的心中充满了希望，相信他们一定能实现他们的目标，创造属于他们的传奇。
