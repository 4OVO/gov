# 第十八章 培训机构邀请

周日上午，林默和苏梦一起去图书馆学习。他们刚坐下，就看到中公教育的王老师走了过来。

"林默，苏梦，你们好，"王老师说道，"恭喜你们在模考中取得了优异的成绩，林默考了86分，苏梦考了78分，真是太厉害了。"

"谢谢王老师，"林默说道。

"林默，苏梦，我们中公教育想再次邀请你们参加我们的公考培训课程，担任助教，"王老师说道，"这次我们可以给你们更高的待遇，而且可以灵活安排时间，不影响你们的备考。"

"王老师，我们现在的首要目标是考上公务员，"林默说道，"所以暂时不想考虑其他事情。"

"没关系，"王老师说道，"我们可以等你们考上公务员之后再邀请你们。对了，我们中公教育想和你们合作，把你们的行测和申论技巧整理成课程，分享给更多的同学。我们可以提供平台和资源，你们提供技巧和经验，课程收入我们可以按照7:3的比例分成，你们拿7成。"

林默想了想："7:3的比例确实很诱人，但是我们现在真的没有时间开发课程。等我们考上公务员之后，再考虑合作的事情吧。"

"好的，"王老师说道，"我们等你们考上公务员之后，再谈合作的事情。这是我的名片，你们考上公务员之后可以随时联系我。"

王老师离开了，林默和苏梦继续学习。他们觉得培训机构的邀请是对他们的认可，但是现在他们的首要目标是考上公务员，所以暂时不想考虑其他事情。

中午，林默和苏梦一起去食堂吃饭。吃饭时，苏梦问道："林默，你觉得我们考上公务员之后，应该和培训机构合作吗？"

"可以考虑，"林默说道，"我们可以把花生十三的技巧分享给更多的同学，帮助他们考上公务员。而且，7:3的分成比例确实很诱人，我们可以用这笔钱做一些有意义的事情。"

"比如呢？"苏梦问道。

"比如成立一个公考公益基金，帮助那些家庭困难的同学备考公务员，"林默说道，"或者组织一些免费的公考培训课程，帮助更多的同学考上公务员。"

"这个想法很好，"苏梦说道，"我们考上公务员之后，就成立一个公考公益基金，帮助那些家庭困难的同学备考公务员。"

"好，"林默说道，"我们考上公务员之后，就成立一个公考公益基金。"

吃完饭，两人回到图书馆，继续学习。他们刚坐下，就看到华图教育的李老师走了过来。

"林默，苏梦，你们好，"李老师说道，"恭喜你们在模考中取得了优异的成绩，林默考了86分，苏梦考了78分，真是太厉害了。"

"谢谢李老师，"林默说道。

"林默，苏梦，我们华图教育想再次邀请你们参加我们的公考培训课程，担任助教，"李老师说道，"这次我们可以给你们更高的待遇，而且可以灵活安排时间，不影响你们的备考。"

"李老师，我们现在的首要目标是考上公务员，"林默说道，"所以暂时不想考虑其他事情。"

"没关系，"李老师说道，"我们可以等你们考上公务员之后再邀请你们。对了，我们华图教育想和你们合作，把你们的行测和申论技巧整理成课程，分享给更多的同学。我们可以提供平台和资源，你们提供技巧和经验，课程收入我们可以按照8:2的比例分成，你们拿8成。"

林默想了想："8:2的比例确实很诱人，但是我们现在真的没有时间开发课程。等我们考上公务员之后，再考虑合作的事情吧。"

"好的，"李老师说道，"我们等你们考上公务员之后，再谈合作的事情。这是我的名片，你们考上公务员之后可以随时联系我。"

李老师离开了，林默和苏梦继续学习。他们觉得越来越多的培训机构注意到了他们，这是对他们的认可，但是现在他们的首要目标是考上公务员，所以暂时不想考虑其他事情。

下午，林默和苏梦在图书馆学习时，遇到了粉笔教育的张老师。张老师也想邀请他们参加粉笔教育的公考培训课程，担任助教，并且想和他们合作开发课程，开出了9:1的分成比例。林默和苏梦同样以备考公务员为由，拒绝了张老师的邀请，说等考上公务员之后再考虑合作的事情。

离开图书馆时，苏梦说道："林默，今天有三个培训机构的老师来邀请我们，而且分成比例一个比一个高，看来我们真的很受关注。"

"是的，"林默说道，"这说明我们的行测和申论技巧很有用，得到了培训机构的认可。"

"那我们考上公务员之后，应该和哪个培训机构合作呢？"苏梦问道。

"我们可以选择一个最适合我们的培训机构合作，"林默说道，"或者我们自己成立一个公考培训工作室，自己开发课程，这样可以更好地控制课程质量，帮助更多的同学考上公务员。"

"自己成立公考培训工作室？"苏梦惊讶地说道，"这是个不错的想法。"

"是的，"林默说道，"我们可以自己成立一个公考培训工作室，邀请一些志同道合的同学加入，一起开发课程，帮助更多的同学考上公务员。"

"好，"苏梦说道，"我们考上公务员之后，就成立一个公考培训工作室。"

回到宿舍，林默躺在床上，回想着今天培训机构的邀请。他觉得这是对他的认可，但是现在他的首要目标是考上公务员，所以暂时不想考虑其他事情。他决定，等考上公务员之后，再考虑和培训机构合作或者成立自己的公考培训工作室的事情。

林默看着自己的未来规划，心中充满了希望。他知道，有了重生的机会和花生十三的技巧，他一定能实现自己的目标，创造属于自己的传奇。
