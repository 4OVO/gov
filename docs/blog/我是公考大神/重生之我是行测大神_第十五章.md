# 第十五章 苏梦的疑惑

周四上午，林默和苏梦一起去图书馆学习。他们刚坐下，苏梦就问道："林默，我一直有一个疑惑，你为什么会这么了解花生十三的技巧？"

林默愣了一下，然后说道："我之前在网上看到他的课程，觉得很有用，就跟着学了。"

"可是花生十三的课程是在2021年才开始走红的，"苏梦说道，"现在是2018年，你怎么会知道他的课程？"

林默被这个问题问得有点措手不及，他想了想："我是在一个论坛上看到有人分享他的课程，所以就知道了。"

"论坛？"苏梦问道，"什么论坛？"

"就是一个公考论坛，"林默说道，"具体名字我忘了。"

苏梦点了点头："哦，原来如此。"

林默知道苏梦对他的回答不太满意，但是他不能告诉苏梦他是重生的，从未来知道花生十三的课程，所以只能用这个理由来解释。

中午，林默和苏梦一起去食堂吃饭。吃饭时，苏梦又问道："林默，你觉得花生十三的技巧真的很有用吗？"

"是的，"林默说道，"花生十三的技巧都是经过实战检验的，特别适合考试。我用他的技巧，行测成绩提高了很多。"

"那你觉得我们能考上公务员吗？"苏梦问道。

"能，"林默说道，"只要我们按照计划认真备考，一定能考上公务员。"

"我相信你，"苏梦说道，"和你在一起学习，我觉得很开心，也很有信心。"

林默笑了笑："我也是，和你在一起学习，我觉得很开心。"

吃完饭，两人回到图书馆，继续学习。林默给苏梦讲解了花生十三的行测和申论技巧，苏梦听得很认真，很快就掌握了这些技巧。

下午，林默和苏梦在图书馆学习时，遇到了他们的同班同学李明。李明看到他们在学习，就走了过来。

"林默，苏梦，你们在学习啊，"李明说道，"我听说你们行测学得很好，想请教一下你们。"

"好啊，"林默说道，"有什么问题你尽管问。"

李明拿出一道数量关系题："林默，这道题我总是做不好，你能教我怎么做吗？"

林默看了看题目："甲、乙两人同时从A地出发到B地，甲的速度是每小时6公里，乙的速度是每小时4公里。甲到达B地后立即返回，在离B地2公里处与乙相遇。问A、B两地的距离是多少？"

林默说道："这道题应该用行程问题的相遇问题来算，甲的速度是每小时6公里，乙的速度是每小时4公里，所以甲和乙的速度比是3:2。在相同的时间内，甲和乙的路程比也是3:2。甲到达B地后立即返回，在离B地2公里处与乙相遇，所以甲比乙多走了2×2=4公里。根据路程比3:2，甲比乙多走了1份，所以1份是4公里，甲走了3份是12公里，乙走了2份是8公里。A、B两地的距离是甲走的路程减去2公里，即12-2=10公里。"

"哇，这方法真的很快，"李明惊讶地说道，"比我之前的方法快多了。"

"是的，"林默说道，"花生十三总结了很多数量关系的秒杀技巧，掌握了这些技巧，做题速度会快很多。"

接下来，林默给李明讲解了花生十三的数量关系秒杀技巧，李明听得很认真，很快就掌握了这些技巧。

李明离开后，苏梦说道："林默，你真的很厉害，什么题都会做。"

"我只是掌握了一些技巧而已，"林默说道，"只要你掌握了这些技巧，你也会变得很厉害。"

"我觉得你不仅掌握了技巧，还很有耐心，"苏梦说道，"你总是很耐心地教我和其他同学，从来不会嫌我们笨。"

"因为我知道学习行测和申论的过程很不容易，"林默说道，"我也经历过这个过程，所以能理解你们的感受。"

苏梦点了点头："你真是个好人。"

林默笑了笑，没有说话。他知道苏梦对他有好感，但是他现在不能向苏梦表白，因为他的首要目标是考上公务员，等考上公务员之后，他会向苏梦表白。

离开图书馆时，苏梦说道："林默，今天的学习收获很大，我觉得我的行测和申论技巧提高了很多。"

"我也觉得今天的学习收获很大，"林默说道，"我们一起努力，争取在国考中取得好成绩。"

"好，"苏梦说道，"我们一起努力，争取都考上公务员。"

林默看着苏梦，心中涌起一股暖流。他知道，这一次，他一定能和苏梦一起考上公务员，创造属于自己的幸福生活。

回到宿舍，林默躺在床上，回想着今天和苏梦的相处，觉得很开心。他知道苏梦对他有好感，但是他现在不能向苏梦表白，因为他的首要目标是考上公务员，等考上公务员之后，他会向苏梦表白。

林默看着自己的未来规划，心中充满了希望。他知道，有了重生的机会和花生十三的技巧，他一定能实现自己的目标，创造属于自己的传奇。
