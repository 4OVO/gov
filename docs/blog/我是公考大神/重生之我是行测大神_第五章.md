# 第五章 开始备考

周一早上，林默按照计划去上课。他的专业课是《市场营销》，老师在讲台上讲解营销策略，林默却在下面看行测教材。他知道，这些专业课对他来说已经不重要了，他的目标是考上公务员，所以他把所有的精力都放在了行测和申论的学习上。

下午三点，林默准时来到图书馆，苏梦已经在那里等他了。她穿着一件粉色的连衣裙，看起来更加漂亮了。

"今天我们学申论吧，"林默说道，"申论是公务员考试的重要组成部分，包括归纳概括、综合分析、提出对策、贯彻执行和申发论述五个部分。"

苏梦点了点头："好，我申论完全不知道从哪里开始学。"

林默翻开申论教材，找到一道归纳概括题："你看这道题，题干要求概括材料中反映的问题。材料主要讲了大学生就业难的问题，包括就业观念不正确、就业能力不足、就业市场供需失衡等。"

苏梦看了看题目，想了想："我觉得应该从就业观念、就业能力、就业市场三个方面来概括。"

"对，"林默说道，"这就是归纳概括题的基本方法，从材料中提取关键词，然后分类概括。花生十三总结了申论的常见题型和解题技巧，掌握了这些技巧，申论就变得简单了。"

接下来，林默给苏梦讲解了花生十三的申论技巧，包括归纳概括的关键词提取、综合分析的辩证思维、提出对策的针对性、贯彻执行的格式要求、申发论述的结构安排等。苏梦学得很认真，很快就掌握了这些技巧。

"林默，申论需要写很多字，我写字很慢，怎么办？"苏梦问道。

"没关系，"林默说道，"申论的字不需要很漂亮，只要工整清晰就行。你可以每天练习写申论，提高写字速度。"

"那你每天都写申论吗？"苏梦问道。

"是的，"林默说道，"我每天都会写一篇申论，保持手感。"

"那我也应该这样做，"苏梦说道，"从明天开始，我每天写一篇申论。"

林默点了点头："好，这样很好。"

两人一直学到晚上六点，然后一起去食堂吃饭。吃饭时，苏梦问道："林默，你知道吗？我最近发现，和你在一起学习很开心。"

林默笑了笑："我也是，和你在一起学习很开心。"

"那我们以后一直在一起学习吧，"苏梦说道，"直到考上公务员。"

"好，"林默说道，"我们一直在一起学习，直到考上公务员。"

吃完饭，两人回到图书馆，继续学习。林默给苏梦讲解了花生十三的行测套题技巧，包括时间分配、答题顺序、涂卡技巧等。苏梦学得很认真，很快就掌握了这些技巧。

"林默，你说我们每天学习这么久，会不会太累？"苏梦问道。

"不会，"林默说道，"我们每天学习八个小时，中间有休息时间，这样的学习强度是合理的。而且，学习行测和申论其实很有意思，掌握了技巧之后，做题就像玩游戏一样。"

"你说得对，"苏梦说道，"我现在觉得做题越来越有意思了，特别是用了花生十三的技巧之后，做题速度快了很多，正确率也提高了。"

晚上八点，林默和苏梦结束了学习。他们一起走出图书馆，苏梦问道："林默，你明天有什么安排？"

"明天还是一样，"林默说道，"早上上课，下午来图书馆学习。"

"好，"苏梦说道，"我明天下午三点在图书馆等你。"

"好，"林默说道，"明天见。"

苏梦笑着点了点头，转身离开了。林默看着她的背影，心中涌起一股暖流。他知道，这一次，他一定能和苏梦在一起，创造属于他们的幸福生活。

回到宿舍，林默躺在床上，回想着今天和苏梦的相处，觉得很开心。他拿出笔记本，开始总结今天的学习内容，然后制定明天的学习计划。

突然，他的手机响了，是他的好朋友陈阳打来的。

"林默，你最近在忙什么？"陈阳问道，"怎么总是见不到你？"

"我在备考公务员，"林默说道，"每天都在图书馆学习。"

"备考公务员？"陈阳惊讶地说道，"你不是要找工作吗？怎么突然要考公务员了？"

"我觉得公务员稳定，福利待遇好，"林默说道，"所以决定考公务员。"

"那你有把握吗？"陈阳问道。

"有，"林默说道，"我掌握了一些行测和申论的技巧，应该能考上。"

"那好吧，"陈阳说道，"祝你成功。对了，周末我们一起去打篮球吧，放松一下。"

"好，"林默说道，"周末一起去打篮球。"

挂断电话，林默躺在床上，想着陈阳。陈阳是他的大学同学，也是他最好的朋友，前世的他，陈阳毕业后去了一家互联网公司工作，后来成为了一名产品经理。这一次，林默希望能和陈阳一起考上公务员，或者一起创业。

林默拿出笔记本，在未来规划中增加了一条：帮助陈阳考上公务员，或者和他一起创业。

林默看着自己的未来规划，心中充满了希望。他知道，有了重生的机会和花生十三的技巧，他一定能实现自己的目标，创造属于自己的传奇。
