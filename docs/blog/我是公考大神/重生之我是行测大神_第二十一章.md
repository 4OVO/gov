# 第二十一章 分享技巧

周三上午，学习小组的第二次活动在图书馆举行。林默和苏梦作为组长，给其他同学讲解了花生十三的资料分析速算技巧。

"同学们，今天我们要学习资料分析的速算技巧，"林默说道，"资料分析是行测中最容易提分的部分，掌握了速算技巧之后，正确率可以达到90%以上。"

林默拿出笔记本，开始给其他同学讲解花生十三的资料分析速算技巧：

**资料分析速算技巧**

**一、截位直除法**
1. **适用场景**：除法运算，尤其是多位数除法
2. **操作方法**：根据选项的差距，对分子和分母进行截位，然后进行直除
3. **示例**：计算631684÷1.072，选项差距较大，可将分母截位为1.07，分子截位为632000，计算632000÷1.07≈589000

**二、估算法**
1. **适用场景**：选项差距较大，不需要精确计算
2. **操作方法**：对算式中的数字进行近似处理，然后进行计算
3. **示例**：计算12345×6789，选项差距较大，可将12345近似为12000，6789近似为7000，计算12000×7000=84000000

**三、特殊分数法**
1. **适用场景**：百分数与分数的转换
2. **操作方法**：将百分数转换为分数，简化计算
3. **示例**：计算12345×16.7%，16.7%≈1/6，所以12345×1/6≈2057.5

其他同学听得很认真，不时点头表示理解。林默讲完之后，苏梦给其他同学讲解了花生十三的申论技巧。

**申论技巧**

**一、归纳概括题**
1. **解题技巧**：提取关键词，分类概括
2. **注意事项**：紧扣材料，不要主观臆断
3. **示例**：概括材料中反映的问题，可从就业观念、就业能力、就业市场三个方面进行概括

**二、综合分析题**
1. **解题技巧**：辩证思维，分析原因和影响
2. **注意事项**：观点明确，逻辑清晰
3. **示例**：分析某政策的影响，可从积极影响和消极影响两个方面进行分析

**三、提出对策题**
1. **解题技巧**：针对问题，提出具体可行的对策
2. **注意事项**：对策要具体，要有可操作性
3. **示例**：针对就业难问题，可提出加强职业教育、完善就业服务体系、鼓励创业等对策

其他同学听得很认真，很快就掌握了这些技巧。

"林默，苏梦，谢谢你们分享这些技巧，"李明说道，"我之前做资料分析题总是做得很慢，正确率也不高，现在用了你们分享的速算技巧，做题速度快了很多，正确率也提高了很多。"

"我也是，"王芳说道，"我之前做申论题总是不知道从哪里开始，现在用了你们分享的申论技巧，做题思路清晰了很多，申论成绩也提高了很多。"

"不用谢，"林默说道，"我们是同学，应该互相帮助。"

"对，"苏梦说道，"我们一起努力，争取都考上公务员。"
