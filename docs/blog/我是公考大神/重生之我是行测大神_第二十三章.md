# 第二十三章 第二次模考

周五上午，林默和苏梦一起去中公教育参加第二次模考。他们已经确定了恋爱关系，现在是时候检验他们的学习成果了。

"林默，你紧张吗？"苏梦问道。

"不紧张，"林默说道，"我们已经掌握了花生十三的行测和申论技巧，只要正常发挥，一定能取得好成绩。"

"我也不紧张，"苏梦说道，"有你在身边，我觉得很安心。"

两人手牵着手，走进了中公教育的教室。他们找了两个相邻的空位坐下，等待模考开始。

模考开始后，林默和苏梦拿到了行测试卷。林默快速浏览了一下试卷，发现试卷的题型和难度与国考相似。他按照花生十三的答题顺序，先做资料分析，然后做言语理解，接着做判断推理，最后做数量关系和常识判断。

林默做资料分析时，使用了花生十三的截位直除法、估算法、特殊分数法等速算技巧，很快就做完了资料分析部分，正确率很高。然后他做言语理解，使用了花生十三的对应关系、结构分析等技巧，也很快就做完了言语理解部分。接着他做判断推理，使用了花生十三的规律总结、关键词法等技巧，同样很快就做完了判断推理部分。最后他做数量关系和常识判断，使用了花生十三的秒杀技巧和积累的知识点，也很快就做完了。

苏梦也按照林默教她的答题顺序和技巧做试卷，她发现自己的做题速度比以前快了很多，正确率也提高了很多。她的心情很愉悦，因为她知道，有林默在身边支持她，她一定能取得好成绩。

模考结束后，林默和苏梦交了试卷，等待成绩公布。成绩公布后，林默看到自己的行测成绩是88分，苏梦的行测成绩是83分，两人都很开心。

"林默，你的行测成绩是88分，好厉害啊，"苏梦说道，"比上次提高了2分。"

"你的行测成绩是83分，也很厉害啊，"林默说道，"比上次提高了5分。"

"这都是你的功劳，"苏梦说道，"如果没有你教我花生十三的技巧，我不可能考这么高的分数。"

"不用谢，"林默说道，"这是你自己努力的结果。而且，我们现在是男女朋友了，互相帮助是应该的。"

苏梦笑了："是的，我们现在是男女朋友了，互相帮助是应该的。"

这时，中公教育的王老师走了过来，对林默和苏梦说道："林默，苏梦，恭喜你们在模考中取得了优异的成绩，林默考了88分，苏梦考了83分，真是太厉害了。"

"谢谢王老师，"林默说道。

"林默，苏梦，我们中公教育想再次邀请你们参加我们的公考培训课程，担任助教，"王老师说道，"这次我们可以给你们更高的待遇，而且可以灵活安排时间，不影响你们的备考。"

"王老师，我们现在的首要目标是考上公务员，"林默说道，"所以暂时不想考虑其他事情。"

"没关系，"王老师说道，"我们可以等你们考上公务员之后再邀请你们。对了，我们中公教育想和你们合作，把你们的行测和申论技巧整理成课程，分享给更多的同学。我们可以提供平台和资源，你们提供技巧和经验，课程收入我们可以按照9:1的比例分成，你们拿9成。"

林默想了想："9:1的比例确实很诱人，但是我们现在真的没有时间开发课程。等我们考上公务员之后，再考虑合作的事情吧。"

"好的，"王老师说道，"我们等你们考上公务员之后，再谈合作的事情。"

王老师离开了，林默和苏梦继续讨论他们的成绩。他们觉得，这次模考的成绩比上次有了很大的提高，这说明他们的学习方法是正确的，只要继续努力，一定能在国考中取得好成绩。

中午，林默和苏梦一起去食堂吃饭。吃饭时，苏梦说道："林默，我们的成绩都有了很大的提高，你觉得我们能在国考中取得好成绩吗？"

"能，"林默说道，"只要我们继续努力，一定能在国考中取得好成绩。我相信，我能考到90分以上，你能考到85分以上。"

"90分以上？85分以上？"苏梦惊讶地说道，"那我们就是真正的行测大神了。"

"是的，"林默说道，"我们就是真正的行测大神。"

吃完饭，两人回到图书馆，继续学习。他们的手牵在一起，感觉很温暖，很幸福。林默给苏梦讲解了花生十三的申论技巧，苏梦听得很认真，很快就掌握了这些技巧。他们的学习效率比以前更高了，因为他们现在不仅是学习伙伴，还是男女朋友，互相鼓励，互相支持。

下午，林默和苏梦在图书馆学习时，遇到了他们的辅导员李老师。李老师看到他们手牵在一起，就走了过来。

"林默，苏梦，你们好，"李老师说道，"我听说你们在模考中取得了优异的成绩，林默考了88分，苏梦考了83分，是吗？"

"是的，李老师，"林默说道，"我们在模考中取得了一些进步。"

"88分和83分，这已经是很高的分数了，"李老师说道，"我相信你们一定能在国考中取得好成绩，考上公务员。"

"谢谢李老师的鼓励，"林默说道，"我们会继续努力的。"

李老师离开了，林默和苏梦继续学习。他们觉得辅导员的鼓励是对他们的认可，更加坚定了他们考上公务员的信心。

离开图书馆时，苏梦说道："林默，今天的模考让我看到了我们的进步，我觉得我们一定能考上公务员。"

"我也觉得我们一定能考上公务员，"林默说道，"只要我们继续努力，一定能实现我们的目标。"

"我们的目标是什么？"苏梦问道。

"我们的目标是考上公务员，然后结婚，一起过上幸福的生活，"林默说道。

"是的，"苏梦说道，"这是我们共同的目标。"

两人手牵着手，一起走回了宿舍。他们的心中充满了希望，相信他们一定能实现他们的目标，创造属于他们的传奇。
