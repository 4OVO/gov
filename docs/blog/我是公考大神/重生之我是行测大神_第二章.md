# 第二章 觉醒金手指

第二天早上七点，林默就醒了。他躺在床上，回想着昨天的经历，仍然有些不敢相信自己重生了，还获得了花生十三的行测技巧。

他试着在脑海中搜索花生十三的资料分析课程，果然，脑海里立刻浮现出了详细的课程内容，包括每一节课的知识点、例题和解析。他甚至能"看到"花生十三在课堂上讲解的画面，听到他的声音。

"这金手指也太厉害了吧，"林默心中感慨道，"简直就像是把花生十三的大脑复制到了我的脑海里。"

他起床洗漱，然后去食堂吃早餐。路上遇到了几个同学，他们都惊讶地看着林默，因为平时林默总是很晚才起床，很少这么早出现在食堂。

"林默，今天怎么这么早？"一个同学问道。

"哦，我要去图书馆学习。"林默笑着回答。

"学习？你不是要考公务员吗？"另一个同学调侃道，"就你那成绩，能考上才怪。"

林默没有生气，只是笑了笑。他知道，前世的自己确实成绩不好，可现在有了花生十三的技巧，他一定能考上公务员，让所有人刮目相看。

吃完早餐，林默来到图书馆，苏梦已经在那里等他了。她穿着一件浅蓝色的T恤，扎着马尾辫，看起来青春靓丽。

"早啊，林默。"苏梦笑着打招呼。

"早，苏梦。"林默在她对面坐下，拿出笔记本和行测教材。

"今天我们学什么？"苏梦问道。

"先学资料分析吧，"林默说道，"资料分析是行测中最容易提分的部分，掌握了技巧之后，正确率可以达到90%以上。"

苏梦点了点头："好，我资料分析总是做得很慢，每次都不够时间。"

林默翻开教材，找到一道资料分析题："你看这道题，2017年全国粮食产量61791万吨，比上年增加166万吨，增产0.3%。其中，夏粮产量14031万吨，增产0.8%；早稻产量3174万吨，减产3.2%；秋粮产量44586万吨，增产0.4%。问2016年全国粮食产量是多少？"

"这道题应该用基期量=现期量/(1+增长率)来算吧，"苏梦说道，"61791除以1.003，等于多少呢？"

她拿出计算器准备计算，林默却拦住了她："不用计算器，用花生十三的截位直除法，把1.003近似看作1，61791除以1就是61791，然后减去61791乘以0.3%，也就是185.37，所以2016年的产量大约是61791-185=61606万吨。"

苏梦惊讶地看着他："你怎么算得这么快？而且结果和正确答案差不多。"

林默笑了笑："这就是花生十三的速算技巧，不用精确计算，通过估算就能快速得出答案。"

"花生十三是谁？"苏梦问道。

"一个很厉害的行测老师，"林默说道，"他的资料分析技巧特别厉害，能让你在短时间内提高做题速度和正确率。"

"那你是怎么知道他的？"苏梦好奇地问道。

林默一时语塞，他总不能说自己是重生的，从未来知道花生十三的吧。他想了想，说道："我之前在网上看到他的课程，觉得很有用，就跟着学了。"

"那你能教我他的技巧吗？"苏梦问道。

"当然可以，"林默说道，"我今天就教你资料分析的速算技巧。"

接下来，林默开始给苏梦讲解花生十三的资料分析速算技巧，包括截位直除法、估算法、特殊分数法、化同法、差分法等。他讲得非常详细，每一种技巧都举了几个例子，苏梦听得很认真，不时点头表示理解。

"这些技巧真的很有用，"苏梦兴奋地说道，"我之前做资料分析题总是要算很久，现在用这些技巧，感觉速度快了很多。"

林默笑着说道："那当然，花生十三的技巧都是经过实战检验的，特别适合考试。"

两人一直学到中午，然后一起去食堂吃饭。吃饭时，苏梦问道："林默，你除了资料分析，其他部分也学了花生十三的技巧吗？"

"是的，"林默说道，"言语理解、判断推理、数量关系和常识判断都学了。"

"那你能不能也教我其他部分的技巧？"苏梦问道。

"当然可以，"林默说道，"我们下午学言语理解吧。"

吃完饭，两人回到图书馆，继续学习。林默给苏梦讲解了花生十三的言语理解技巧，包括逻辑填空的对应关系、片段阅读的结构分析、语句排序的技巧等。苏梦学得很认真，很快就掌握了这些技巧。

"林默，你好像变了一个人，"苏梦突然说道，"以前的你总是很内向，不爱说话，现在不仅话多了，而且变得很自信。"

林默愣了一下，然后笑了："可能是因为我找到了学习行测的方法，有了目标，所以变得自信了吧。"

苏梦点了点头："这样很好，自信的人更容易成功。"

下午四点，林默和苏梦结束了学习。他们一起走出图书馆，阳光洒在他们身上，林默觉得，这是他重生以来最美好的一天。

回到宿舍，林默躺在床上，继续探索脑海中的花生十三行测技巧。他发现，除了技巧之外，脑海里还有花生十三的备考经验和考试心得，甚至还有他的教学方法。

"这金手指简直是为我量身定做的，"林默心中感慨道，"有了这些，我不仅能考上公务员，还能成为一名像花生十三一样的行测老师。"

他突然想到，花生十三是在2021年开始走红的，而现在是2018年，花生十三还没有成名。如果他现在开始分享行测技巧，是不是能比花生十三更早走红？

不过，林默很快就打消了这个念头。他现在的首要目标是考上公务员，改变自己的命运。至于成为行测老师，那是以后的事情。

晚上，林默做了一个梦，梦到自己在国考中取得了行测86分的好成绩，苏梦也考上了公务员，两人一起进入了同一单位工作，过上了幸福的生活。

第二天早上，林默醒来，发现自己嘴角还带着笑容。他知道，这个梦很快就会成为现实。
