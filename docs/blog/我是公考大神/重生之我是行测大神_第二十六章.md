# 第二十六章 国考报名

周一上午，林默和苏梦一起去图书馆学习。他们已经在网络上走红，有很多人在关注他们，但是他们并没有被网络走红冲昏头脑，而是继续努力备考，争取在国考中取得好成绩。

"林默，国考报名开始了，"苏梦说道，"我们应该什么时候报名？"

"国考报名一般在10月中旬开始，持续10天左右，"林默说道，"我们应该在报名开始后的前几天报名，这样如果有问题可以及时修改。"

"那我们今天就报名吧，"苏梦说道，"现在应该已经开始报名了。"

"好，"林默说道，"我们现在就去报名。"

林默和苏梦来到学校的计算机房，打开国家公务员局的网站，开始报名。他们首先注册了账号，然后填写了个人信息，包括姓名、性别、身份证号、学历、专业等。接着，他们选择了报考职位。林默选择了国家税务总局的一个职位，苏梦选择了财政部的一个职位。最后，他们上传了照片，确认了报名信息，完成了报名。

报名完成后，林默和苏梦回到图书馆继续学习。他们的心情很愉悦，因为他们已经完成了国考报名，现在是时候更加努力地备考，争取在国考中取得好成绩。

中午，林默和苏梦一起去食堂吃饭。吃饭时，苏梦说道："林默，我们已经完成了国考报名，你觉得我们能考上我们选择的职位吗？"

"能，"林默说道，"我们选择的职位虽然竞争激烈，但是我们已经掌握了花生十三的行测和申论技巧，只要正常发挥，一定能考上。"

"我相信你，"苏梦说道，"有你在身边支持我，我觉得很有信心。"

"我也相信你，"林默说道，"你是最棒的。"

吃完饭，两人回到图书馆，继续学习。林默给苏梦讲解了花生十三的申论技巧，苏梦听得很认真，很快就掌握了这些技巧。他们的学习效率比以前更高了，因为他们现在不仅是学习伙伴，还是男女朋友，互相鼓励，互相支持。

下午，林默和苏梦在图书馆学习时，遇到了学习小组的其他同学。这些同学看到他们，就走了过来，询问他们的国考报名情况。

"林默，苏梦，你们已经报名了吗？"张阳问道。

"是的，"林默说道，"我们已经报名了。"

"你们选择了什么职位？"李明问道。

"我选择了国家税务总局的一个职位，"林默说道，"苏梦选择了财政部的一个职位。"

"国家税务总局和财政部？"王芳惊讶地说道，"这些职位的竞争很激烈，你们有信心考上吗？"

"有，"林默和苏梦异口同声地说道，"我们已经掌握了花生十三的行测和申论技巧，只要正常发挥，一定能考上。"

其他同学点了点头，然后回到自己的座位上学习。林默和苏梦继续学习，他们觉得学习小组的同学的关心是对他们的鼓励，更加坚定了他们考上公务员的信心。

晚上，林默和苏梦一起去食堂吃饭。吃饭时，苏梦说道："林默，我们已经完成了国考报名，现在应该更加努力地备考，争取在国考中取得好成绩。"

"是的，"林默说道，"我们现在应该制定一个详细的冲刺计划，按照计划认真执行，争取在国考前进一步提高成绩。"

"那我们的冲刺计划是什么？"苏梦问道。

"我们的冲刺计划是这样的，"林默说道，"每天学习行测和申论各四个小时，每周参加一次模考，每天总结错题，每天积累常识和申论素材，同时在微博和微信公众号上分享学习方法和技巧，帮助更多的人提高成绩。"

"好，"苏梦说道，"我们就按照这个冲刺计划执行，争取在国考前进一步提高成绩。"

吃完饭，两人回到图书馆，继续学习。他们制定了详细的冲刺计划，然后按照计划开始学习。林默给苏梦讲解了花生十三的行测和申论技巧，苏梦听得很认真，很快就掌握了这些技巧。

离开图书馆时，苏梦说道："林默，我们已经完成了国考报名，制定了冲刺计划，现在应该更加努力地备考，争取在国考中取得好成绩。"

"是的，"林默说道，"我们要保持初心，不要被网络走红冲昏头脑，继续努力备考，争取在国考中取得好成绩，实现我们的目标。"

"我们的目标是什么？"苏梦问道。

"我们的目标是考上公务员，然后结婚，一起过上幸福的生活，"林默说道，"同时，我们要帮助更多的人考上公务员，让他们也能实现自己的梦想。"

"是的，"苏梦说道，"这是我们共同的目标。"

两人手牵着手，一起走回了宿舍。他们的心中充满了希望，相信他们一定能实现他们的目标，创造属于他们的传奇。
