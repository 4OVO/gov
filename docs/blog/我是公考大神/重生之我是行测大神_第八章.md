# 第八章 判断推理技巧

周四下午三点，林默准时来到图书馆，苏梦已经在那里等他了。她穿着一件紫色的上衣，看起来优雅大方。

"今天我们重点学习判断推理的技巧，"林默说道，"判断推理是行测中最需要逻辑思维的部分，也是最容易拉开差距的部分。掌握了判断推理的技巧，就能在行测考试中占据优势。"

苏梦点了点头："好，我判断推理中的图形推理总是做不好，经常找不到规律。"

林默翻开判断推理教材，找到一道图形推理题："你看这道题，题干中的图形都是由两个部分组成的，而且这两个部分都是对称图形。第一个图形是两个半圆，第二个是两个三角形，第三个是两个矩形，第四个应该是什么？"

苏梦看了看题目，想了想："应该是两个对称图形组成的，选项中只有C选项是两个对称图形。"

"对，"林默说道，"这就是图形推理的对称性规律。花生十三总结了图形推理的常见规律，包括对称性、数量关系、位置变化、样式运算等，掌握了这些规律，图形推理就变得简单了。"

接下来，林默给苏梦讲解了花生十三的判断推理技巧，包括图形推理的规律总结、定义判断的关键词法、类比推理的逻辑关系、逻辑判断的翻译推理等。他讲得非常详细，每一种技巧都举了几个例子，苏梦听得很认真，不时点头表示理解。

"林默，你看这道题，"苏梦指着一道定义判断题说道，"题干是'垂直绿化指的是利用攀援植物向空中生长进行纵向绿化的一种方式，以期达到在有限面积内最大限度地利用空气和阳光来提高绿化的效率。'问下列属于垂直绿化的是？选项是A.爬山虎爬满了墙头和屋顶 B.松树林从山脚一直延伸到山顶 C.利用巴根草匍匐攀援的特性，人们在沙漠中逐渐开拓出一片绿洲 D.在丘陵地区，人们常使用飞机从空中播种的方式对山地进行绿化。"

苏梦看了看题目，想了想："根据定义，垂直绿化是利用攀援植物向空中生长进行纵向绿化的一种方式。选项A.爬山虎爬满了墙头和屋顶，爬山虎是攀援植物，向空中生长，属于垂直绿化。选项B.松树林从山脚一直延伸到山顶，松树不是攀援植物，不属于垂直绿化。选项C.利用巴根草匍匐攀援的特性，巴根草是匍匐攀援，不是向空中生长，不属于垂直绿化。选项D.在丘陵地区，人们常使用飞机从空中播种的方式对山地进行绿化，不是利用攀援植物，不属于垂直绿化。所以正确答案是A。"

"对，"林默说道，"这就是定义判断的关键词法，需要找到定义的关键词，然后逐一排除选项。定义的关键词是'利用攀援植物'、'向空中生长'、'纵向绿化'，所以正确答案是A。"

两人一直学到中午，然后一起去食堂吃饭。吃饭时，苏梦问道："林默，你说判断推理的技巧需要多长时间才能掌握？"

"因人而异，"林默说道，"一般来说，只要每天练习30道判断推理题，一个月就能掌握这些技巧。"

"那我每天都练习30道判断推理题，"苏梦说道，"争取一个月内掌握这些技巧。"

林默点了点头："好，这样很好。"

吃完饭，两人回到图书馆，继续学习。林默给苏梦讲解了花生十三的判断推理常考题型，包括图形推理、定义判断、类比推理、逻辑判断等。苏梦学得很认真，很快就掌握了这些题型的解题技巧。

"林默，你看这道题，"苏梦指着一道类比推理题说道，"题干是'白醋∶消毒'，选项是A.白糖∶调味 B.人参∶滋补 C.热水器∶加热 D.汽油∶去渍。"

苏梦看了看题目，想了想："白醋的主要功能是调味，消毒是它的次要功能。选项A.白糖的主要功能是调味，不是次要功能。选项B.人参的主要功能是滋补，不是次要功能。选项C.热水器的主要功能是加热，不是次要功能。选项D.汽油的主要功能是燃料，去渍是它的次要功能。所以正确答案是D。"

"对，"林默说道，"这就是类比推理的功能对应关系，需要区分主要功能和次要功能。你做得很好。"

"林默，你看这道题，"苏梦指着一道逻辑判断题说道，"题干是'如果下雨，那么地湿。地湿了，所以下雨了。'问这是什么推理错误？"

苏梦看了看题目，想了想："这是充分条件假言推理的错误，肯定后件不能肯定前件。下雨是地湿的充分条件，但地湿不一定是下雨导致的，可能是其他原因，比如有人泼水。"

"对，"林默说道，"这就是逻辑判断的翻译推理，需要掌握充分条件和必要条件的推理规则。你做得很好。"

两人一直学到晚上八点，然后一起走出图书馆。苏梦说道："林默，今天的学习收获很大，我觉得我的判断推理技巧提高了很多。"

林默笑了笑："只要你坚持练习，你的判断推理水平会越来越高，正确率会越来越高。"

"谢谢你，林默，"苏梦说道，"如果没有你，我可能永远都不会掌握这些技巧。"

"不用谢，"林默说道，"我们是朋友，应该互相帮助。"

苏梦笑着点了点头，转身离开了。林默看着她的背影，心中涌起一股暖流。他知道，这一次，他一定能和苏梦在一起，创造属于自己的幸福生活。

回到宿舍，林默躺在床上，回想着今天和苏梦的相处，觉得很开心。他拿出笔记本，开始总结今天的学习内容，然后制定明天的学习计划。

突然，他的手机响了，是中公教育的王老师打来的。

"你好，林默同学，"王老师说道，"我们听说你行测学得很好，特别是判断推理，我们想邀请你参加我们的公考培训课程，担任助教。"

林默愣了一下，然后说道："谢谢邀请，但是我现在正在备考公务员，没有时间担任助教。"

"没关系，"王老师说道，"我们可以等你考上公务员之后再邀请你。对了，我们最近有一个判断推理的专题讲座，想请你去做嘉宾，分享一下你的学习经验，你有兴趣吗？"

林默想了想，说道："不了，谢谢，我现在有自己的学习计划，就不参加了。"

"好吧，"王老师说道，"如果你有兴趣的话，随时可以联系我们。"

挂断电话，林默躺在床上，想着明天的篮球比赛。他知道，适当的放松对备考很重要，所以他决定明天去打篮球，放松一下心情。

林默看着自己的未来规划，心中充满了希望。他知道，有了重生的机会和花生十三的技巧，他一定能实现自己的目标，创造属于自己的传奇。
