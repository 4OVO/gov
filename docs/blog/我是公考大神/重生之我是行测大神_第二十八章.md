# 第二十八章 调整心态

周三上午，林默和苏梦一起去图书馆学习。他们已经制定了详细的冲刺计划，现在是时候调整心态，应对国考压力了。

"林默，我最近有点紧张，"苏梦说道，"国考越来越近了，我担心自己考不好。"

"苏梦，不要紧张，"林默说道，"我们已经掌握了花生十三的行测和申论技巧，只要正常发挥，一定能考出好成绩。"

"但是我还是很紧张，"苏梦说道，"我担心自己在考试中会犯错误，会发挥失常。"

"紧张是正常的，"林默说道，"但是过度紧张会影响考试发挥。我们需要调整心态，保持冷静，以平常心对待国考。"

"那我们应该怎么调整心态呢？"苏梦问道。

"我们可以通过以下方法调整心态，"林默说道，"第一，适当运动，释放压力；第二，听音乐，放松心情；第三，与朋友交流，分享感受；第四，保持充足的睡眠，养精蓄锐。"

"好，"苏梦说道，"我们今天下午就去运动，释放压力。"

"好，"林默说道，"我们今天下午去打篮球，释放压力。"

下午，林默和苏梦来到学校的篮球场，开始打篮球。林默已经很久没有打篮球了，但是他很快就找到了状态。他在球场上奔跑、跳跃、投篮，感觉很放松，很愉快。苏梦也在旁边为他加油，感觉心情也变得很愉快。

打篮球结束后，林默和苏梦坐在场边休息。苏梦说道："林默，打篮球真的很放松，我感觉心情好多了。"

"是的，"林默说道，"适当的运动可以释放压力，调整心态，对备考很有帮助。"

"那我们以后每天都来打篮球，"苏梦说道，"这样可以保持良好的心态。"

"好，"林默说道，"我们以后每天下午都来打篮球，释放压力，调整心态。"

晚上，林默和苏梦一起去食堂吃饭。吃饭时，苏梦说道："林默，我感觉现在的心态好多了，不再那么紧张了。"

"那就好，"林默说道，"保持良好的心态是备考的重要环节，只有心态好，才能在考试中正常发挥。"

"我会保持良好的心态，"苏梦说道，"争取在国考中正常发挥，取得好成绩。"

"我相信你，"林默说道，"你是最棒的。"

吃完饭，两人回到图书馆，继续学习。他们总结了当天的错题，分析了错误原因，然后在微博和微信公众号上分享了学习方法和技巧。他们的粉丝数量不断增加，越来越多的人开始关注他们，向他们请教学习方法和技巧。他们耐心地回复每一条私信和评论，分享花生十三的行测和申论技巧，帮助更多的人提高成绩。

离开图书馆时，苏梦说道："林默，今天的学习很充实，我感觉心态好多了，不再那么紧张了。"

"是的，"林默说道，"保持良好的心态是备考的重要环节，只有心态好，才能在考试中正常发挥。"

"我们的目标是什么？"苏梦问道。

"我们的目标是考上公务员，然后结婚，一起过上幸福的生活，"林默说道，"同时，我们要帮助更多的人考上公务员，让他们也能实现自己的梦想。"

"是的，"苏梦说道，"这是我们共同的目标。"

两人手牵着手，一起走回了宿舍。他们的心中充满了希望，相信他们一定能实现他们的目标，创造属于他们的传奇。
