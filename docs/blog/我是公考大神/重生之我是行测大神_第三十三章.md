# 第三十三章 考完放松

国考结束后，林默和苏梦终于可以放松心情，享受一段没有压力的时光。他们决定和学习小组的同学一起聚餐，庆祝国考顺利结束，同时回顾这段难忘的备考经历。

**聚餐准备**

周六晚上，林默和苏梦来到学校附近的一家餐厅，学习小组的其他同学已经在那里等他们了。张阳、李明、王芳等人看到他们，纷纷热情地打招呼，大家的脸上都洋溢着轻松愉快的笑容。

**回顾备考**

点餐结束后，大家开始回顾这段备考的日子。张阳感慨道：“这段时间真的很辛苦，每天早起学习，晚上还要总结错题，但是现在回想起来，觉得很充实，也很值得。”李明接着说：“是啊，要不是林默和苏梦建立学习小组，分享花生十三的技巧，我们可能还在盲目备考，效率低很多。”王芳也说道：“没错，花生十三的技巧真的很实用，尤其是资料分析的速算技巧，帮我节省了很多时间。”

林默和苏梦听了，心里很开心。林默说道：“其实我们也很感谢大家，和你们一起学习，我们也收获了很多。这段经历让我们明白了，一个人可以走得很快，但一群人可以走得更远。”苏梦也说道：“是的，我们一起努力，一起进步，这种感觉真的很好。”

**分享考试感受**

大家开始分享各自的考试感受。张阳说：“我行测考得还可以，资料分析用了林默教的截位直除法，做得很快，正确率也很高。申论大作文写的是‘数字经济与乡村振兴’，结构很清晰，感觉应该能拿高分。”李明说：“我也是，申论用了苏梦教的关键词提取技巧，归纳概括题做得很顺，答案很全面。”王芳说：“我行测数量关系用了林默教的秒杀技巧，做对了很多以前不会的题目，真的很感谢你们。”

林默和苏梦听了，心里很欣慰。他们觉得，能够帮助同学提高成绩，是一件很有意义的事情。他们相信，大家的努力一定会有回报，一定能考上自己理想的职位。

**放松活动**

聚餐结束后，大家决定一起去KTV唱歌，放松心情。在KTV里，大家唱着欢快的歌曲，释放着备考期间的压力。林默唱了一首《海阔天空》，歌词“原谅我这一生不羁放纵爱自由，也会怕有一天会跌倒，背弃了理想，谁人都可以，哪会怕有一天只你共我”引起了大家的共鸣，大家一起跟着唱，场面很热闹。

苏梦唱了一首《小幸运》，歌词“原来你是我最想留住的幸运，原来我们和爱情曾经靠得那么近”让林默很感动，他知道，苏梦是在唱给他听。唱完后，苏梦走到林默身边，握住他的手，两人相视而笑，心中充满了幸福。

**未来计划**

晚上十点，大家结束了KTV活动，各自回宿舍休息。林默和苏梦一起走在回宿舍的路上，月光洒在他们身上，感觉很温馨。苏梦说道：“林默，这段备考的日子虽然辛苦，但我觉得很幸福，因为有你在身边。”林默说道：“我也是，有你在身边，再苦再累也值得。”

苏梦问道：“林默，国考结束了，你接下来有什么计划？”林默说道：“我想先休息几天，然后开始准备面试，同时继续在微博和微信公众号上分享学习方法和技巧，帮助更多的人提高成绩。你呢？”苏梦说道：“我和你一样，先休息几天，然后准备面试，同时帮助你管理微博和微信公众号。”

林默说道：“对了，我们之前说过，考上公务员之后要成立一个公考公益基金，帮助家庭困难的同学备考，我们可以开始准备了。”苏梦说道：“好啊，我们可以先制定一个详细的计划，包括基金的资金来源、资助对象、资助标准等。”

两人一边走一边讨论未来的计划，他们的心中充满了希望，相信未来会更加美好。

**回到宿舍**

回到宿舍后，林默和苏梦各自洗漱完毕，躺在床上。林默给苏梦发了一条微信：“今天很开心，和大家一起庆祝国考结束，感觉这段经历很值得。”苏梦回复：“我也是，有你在身边，我觉得很幸福。晚安，明天见。”林默回复：“晚安，明天见。”

躺在床上，林默回想起这段备考的日子，从重生到觉醒金手指，从建立学习小组到国考结束，一切仿佛像一场梦，但又是那么真实。他知道，这只是人生的一个阶段，未来还有更多的挑战和机遇在等待着他。但他相信，只要有苏梦在身边，有朋友们的支持，他一定能克服所有困难，实现自己的梦想。

苏梦也躺在床上，回想起这段备考的日子，从遇到林默到成为男女朋友，从学习花生十三的技巧到国考结束，她觉得自己很幸运，能够遇到林默，能够和他一起为了目标努力。她相信，未来的日子会更加美好，他们会一起考上公务员，一起结婚，一起过上幸福的生活。
