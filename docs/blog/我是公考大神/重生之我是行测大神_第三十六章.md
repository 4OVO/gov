# 第三十六章 行测大神的诞生

国考成绩公布后，林默和苏梦的高分迅速在网络上引起了轰动。林默的行测92分、总分170分，苏梦的行测85分、总分157分，这样的成绩在国考中堪称顶级，两人因此被网友称为“行测大神”。

**媒体关注**

成绩公布后的第二天，就有多家媒体联系林默和苏梦，希望采访他们的备考经历和学习方法。其中包括当地的电视台、报纸，以及一些知名的教育类网站。林默和苏梦接受了部分媒体的采访，分享了他们的备考经历和花生十三的学习技巧。

在接受采访时，林默说：“我的成功主要归功于花生十三的行测技巧，尤其是资料分析的速算技巧，帮我节省了大量时间，提高了正确率。同时，苏梦的支持和鼓励也是我成功的重要因素。”苏梦也说：“是的，我们一起学习，互相鼓励，这种合作精神让我们事半功倍。”

**培训机构邀请**

多家公考培训机构也向林默和苏梦发出了邀请，希望他们能成为机构的讲师，开出的年薪高达几十万。中公教育的王老师说：“林默和苏梦的成绩非常优秀，他们的学习方法和技巧很有推广价值，我们希望能与他们合作，把这些技巧分享给更多的考生。”华图教育的李老师也说：“我们可以提供更灵活的工作时间和更高的待遇，希望他们能考虑加入我们。”

面对这些邀请，林默和苏梦经过认真考虑后，决定暂时不接受任何培训机构的邀请。林默说：“我们现在的首要任务是准备面试，希望能一次成功考上公务员。至于未来是否与培训机构合作，我们会在考上公务员后再考虑。”

**网络走红**

林默和苏梦的微博和微信公众号粉丝数量急剧增长，从几万人增长到了几十万人。他们发布的备考经验和学习技巧被广泛转发，成为了公考领域的“网红”。很多网友在评论区留言，表达对他们的敬佩和感谢。有网友说：“林默和苏梦的故事太励志了，他们的学习方法和技巧很实用，我要向他们学习。”还有网友说：“花生十三的技巧果然厉害，我也要开始学习这些技巧，希望能像他们一样成功。”

**学习小组的骄傲**

学习小组的其他同学也为林默和苏梦的成功感到骄傲。张阳说：“林默和苏梦是我们学习小组的骄傲，他们的成功证明了花生十三技巧的有效性，也证明了努力的重要性。”李明说：“我现在更有信心了，相信在他们的帮助下，我也能考上公务员。”王芳说：“我要以他们为榜样，更加努力地备考，争取下次也能取得好成绩。”

**公益基金的进展**

林默和苏梦的公考公益基金也得到了更多人的关注和支持。很多企业和个人主动联系他们，希望能为基金捐款。林默说：“我们非常感谢大家的支持，这些捐款将用于帮助更多家庭困难的公考考生，让他们也能享受到优质的备考资源。”苏梦也说：“我们会定期公布基金的使用情况，确保每一分钱都用到刀刃上。”

**面试准备的专注**

尽管受到了很多关注，林默和苏梦并没有被这些外界因素干扰，而是更加专注于面试准备。他们每天花费大量时间学习面试技巧，参加面试模拟训练，互相点评，不断提高面试能力。林默说：“面试是公务员考试的重要环节，我们不能因为笔试成绩高就掉以轻心，必须认真准备，争取一次成功。”苏梦也说：“是的，面试考查的是综合能力，我们需要在语言表达、逻辑思维、应急反应等方面都做好准备。”

**回顾备考关键节点**

在面试准备的间隙，林默和苏梦会回顾备考过程中的关键节点。林默说：“现在回想起来，重生后的觉醒金手指是我成功的起点，让我接触到了花生十三的技巧。然后建立学习小组，与苏梦一起学习，互相鼓励，这些都是我成功的重要因素。”苏梦说：“是的，从第一次模考的紧张，到后来的逐渐自信，再到国考的从容应对，我们经历了很多，但每一步都走得很踏实。”

**感谢花生十三**

林默和苏梦始终对花生十三充满感激。林默说：“花生十三的技巧不仅帮助我提高了成绩，更重要的是改变了我的学习方法和思维方式，让我学会了如何高效学习。”苏梦说：“是的，申论的学习技巧让我不再害怕写申论，反而觉得申论很有意思，这是我之前从未想过的。”

**未来的规划**

对于未来，林默和苏梦有清晰的规划。他们希望考上公务员后，继续经营微博和微信公众号，分享学习方法和技巧，帮助更多的考生；同时，扩大公考公益基金的规模，资助更多家庭困难的考生；此外，他们还计划在业余时间编写公考辅导资料，将花生十三的技巧与自己的学习经验结合起来，为考生提供更全面、更实用的备考资源。

**结语**

林默和苏梦的成功告诉我们，正确的学习方法和技巧是成功的关键，但努力和坚持同样重要。花生十三的行测和申论技巧为他们提供了高效的学习工具，而他们的努力和坚持则让这些工具发挥了最大的作用。

行测大神的诞生不是偶然，而是技巧、努力和坚持的结晶。愿所有公考考生都能找到适合自己的学习方法和技巧，通过努力实现自己的公务员梦想。
