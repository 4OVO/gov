# 《重生之我是行测大神》小说大纲

## 一、基本信息
- **书名**：重生之我是行测大神
- **作者**：[花生十三小迷弟]
- **字数**：二十万字
- **类型**：都市重生、公考题材
- **核心卖点**：结合花生十三的考试经历，重生后利用行测技巧逆袭，成为公考神话

## 二、故事梗概
主角林默是一名普通上班族，多次参加公考失败，最终在加班时猝死。意外重生到2018年大学毕业前夕，获得了未来花生十三的行测知识和技巧，加上自己的现代记忆，开启了一段逆袭之路。他不仅在行测考试中创造神话，还收获了爱情和友情，同时揭秘了公考培训行业的内幕，成为了一代公考名师。

## 三、主要人物
1. **林默**：主角，重生前多次公考失败，重生后利用花生十三的技巧成为行测大神
2. **苏梦**：女主角，林默的大学同学，聪明漂亮，与林默一起备考
3. **陈阳**：林默的好友，热血青年，喜欢打篮球，后来成为林默的创业伙伴
4. **王老师**：传统公考培训机构的老师，一开始看不起林默，后来被其折服
5. **李校长**：某公考培训机构校长，试图挖角林默
6. **花生十三**：未来的公考名师，其知识和技巧被林默继承
