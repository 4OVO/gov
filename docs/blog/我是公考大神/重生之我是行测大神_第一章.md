# 第一章 猝死重生

凌晨三点，办公室的灯光还亮着。林默揉了揉发酸的眼睛，看着电脑屏幕上的公考资料，心里一片迷茫。

这已经是他第三次备考公务员了。前两次都是行测拖了后腿，特别是资料分析部分，总是做得又慢又错。他听说花生十三的资料分析课很厉害，特意花了半个月工资买了课程，可还没看完一半，就被公司派去出差，回来后又要加班赶项目。

"要是能重来一次就好了，"林默自言自语道，"我一定好好学行测，再也不熬夜加班了。"

突然，一阵剧烈的胸痛袭来，林默眼前一黑，倒在了办公桌上。

等他再次睁开眼睛时，发现自己躺在大学宿舍的床上。阳光透过窗户洒进来，床头的日历显示着2018年6月15日。

"这是...我大学毕业前的最后一个月？"林默揉了揉眼睛，不敢相信自己的眼睛。他记得自己明明在办公室加班，怎么会回到了六年前？

他掀开被子下床，走到镜子前，镜子里的自己年轻了六岁，还是大学时的模样。他捏了捏自己的脸，疼得龇牙咧嘴，这不是梦！

"我重生了！"林默激动地喊道。

宿舍里的其他三个室友都不在，应该是去图书馆或者找工作了。林默坐在书桌前，回忆着自己的前世。

前世的他，大学毕业后先是在一家私企工作，每天加班到凌晨，工资却只有六千多。后来听说公务员稳定，福利待遇好，就开始备考。可前两次都因为行测分数太低而失败，第三次备考时猝死在了办公室。

现在重生了，他有了重来一次的机会。这一次，他一定要考上公务员，改变自己的命运。

突然，他的脑海里闪过一些陌生的记忆。那是关于花生十三的行测技巧，包括资料分析的速算方法、言语理解的解题技巧、判断推理的逻辑思路、数量关系的秒杀公式，甚至还有常识判断的积累方法。

"这是...金手指？"林默惊讶地发现，自己的脑海里仿佛有一个行测知识库，里面包含了花生十三所有的技巧和方法。

他试着回忆资料分析的速算技巧，果然，脑海里立刻浮现出了花生十三的"截位直除法"、"估算法"、"特殊分数法"等多种速算方法，还有详细的例题和解析。

林默激动得手都发抖了。有了这些技巧，他的行测成绩一定会突飞猛进。

这时，宿舍门被推开了，一个穿着白色连衣裙的女孩走了进来。她有着一头乌黑的长发，皮肤白皙，眼睛明亮动人，正是林默的大学同学苏梦。

"林默，你怎么没去图书馆？"苏梦惊讶地问道，"马上就要毕业了，大家都在忙着找工作或者备考，你怎么还在宿舍？"

林默看着眼前的苏梦，心跳加速。前世的他，一直暗恋苏梦，可因为自卑，从来不敢表白。后来苏梦考上了公务员，而他却一直没能上岸，两人逐渐失去了联系。

"我...我在想事情。"林默结结巴巴地说道。

苏梦笑了笑，坐在他对面的床上："你是不是也在纠结是找工作还是考公务员？我听说你之前在准备公务员考试。"

林默点了点头："是啊，我想考公务员，可行测总是学不好。"

"没关系，我们可以一起备考。"苏梦鼓励道，"我也在准备公务员考试，有什么不懂的我们可以一起讨论。"

林默看着苏梦，心中涌起一股暖流。这一次，他不仅要考上公务员，还要和苏梦在一起。

"好，我们一起备考。"林默坚定地说道。

苏梦笑着点了点头，从书包里拿出一本行测教材："我刚买了一本教材，我们一起看看吧。"

林默接过教材，翻到资料分析部分。看着上面的例题，他立刻用花生十三的技巧算出了答案，比教材上的解析快了好几倍。

"苏梦，你看这道题，"林默指着一道资料分析题说道，"用截位直除法，把12345截成12300，除以1.23，很快就能算出答案是10000。"

苏梦惊讶地看着他："你怎么这么快就算出来了？我还在看题目呢。"

林默笑了笑："我之前看了一些技巧视频，学到了一些速算方法。"

"真厉害，"苏梦崇拜地说道，"那以后你要教我这些技巧哦。"

"没问题，"林默说道，"我们一起努力，争取都考上公务员。"

苏梦点了点头，两人开始一起学习行测。林默发现，有了花生十三的技巧，行测变得简单了很多。他不仅能快速解题，还能给苏梦讲解各种技巧，两人的关系也越来越亲密。

不知不觉，天已经黑了。苏梦看了看手表，说道："时间不早了，我要回宿舍了。明天我们再一起学习吧。"

"好，"林默说道，"明天早上八点，图书馆见。"

苏梦笑着离开了宿舍。林默坐在书桌前，看着窗外的星空，心中充满了希望。

这一次，他有了重生的机会，有了花生十三的行测技巧，还有苏梦的陪伴。他相信，自己一定能考上公务员，改变自己的命运。

"重生之我是行测大神，"林默喃喃自语道，"这一次，我要创造神话。"
