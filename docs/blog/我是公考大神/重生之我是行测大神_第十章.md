# 第十章 常识判断积累

周六上午十点，林默准时来到学校篮球场，陈阳已经在那里等他了。陈阳穿着一件黑色的篮球服，看起来精神抖擞。

"林默，你来了，"陈阳说道，"我们已经开始了，快来加入吧。"

林默点了点头，换上篮球鞋，加入了比赛。他已经很久没有打篮球了，感觉有些生疏，但很快就找到了状态。他在球场上奔跑、跳跃、投篮，仿佛回到了大学时代。

比赛结束后，林默和陈阳坐在场边休息。陈阳问道："林默，你最近备考公务员怎么样了？"

"很好，"林默说道，"我掌握了一些行测和申论的技巧，应该能考上公务员。"

"那太好了，"陈阳说道，"祝你成功。对了，你有没有想过，考上公务员之后做什么？"

"我想先在单位好好工作，"林默说道，"然后利用业余时间做一些公考培训的工作，帮助更多的人考上公务员。"

"公考培训？"陈阳问道，"你想当老师吗？"

"是的，"林默说道，"我觉得行测和申论的技巧很有用，应该让更多的人知道这些技巧，帮助他们考上公务员。"

"那如果我想考公务员，你能教我吗？"陈阳问道。

"当然可以，"林默说道，"等我考上公务员之后，我就教你。"

陈阳点了点头："好，那我等你考上公务员之后，就开始备考。"

中午，林默和陈阳一起去食堂吃饭。吃完饭，林默回到宿舍休息了一会儿，然后去图书馆学习。苏梦已经在那里等他了。

"林默，你今天上午去哪里了？"苏梦问道。

"我去打篮球了，"林默说道，"和陈阳一起。"

"打篮球？"苏梦惊讶地说道，"我以为你一直在学习呢。"

"适当的放松对备考很重要，"林默说道，"打篮球可以放松心情，提高学习效率。"

"那你今天下午要教我什么？"苏梦问道。

"今天下午我们学习常识判断的积累方法，"林默说道，"常识判断是行测中最需要积累的部分，也是最容易被忽视的部分。掌握了常识判断的积累方法，就能在行测考试中占据优势。"

苏梦点了点头："好，我常识判断总是做得不好，很多知识点都不知道。"

林默翻开常识判断教材，找到一道时政题："你看这道题，2017年10月18日，中国共产党第十九次全国代表大会在北京召开，习近平总书记代表第十八届中央委员会向大会作了题为《决胜全面建成小康社会 夺取新时代中国特色社会主义伟大胜利》的报告。问党的十九大报告指出，中国特色社会主义进入新时代，我国社会主要矛盾已经转化为？选项是A.人民日益增长的物质文化需要同落后的社会生产之间的矛盾 B.人民日益增长的美好生活需要和不平衡不充分的发展之间的矛盾 C.人民日益增长的美好生活需要和落后的社会生产之间的矛盾 D.人民日益增长的物质文化需要和不平衡不充分的发展之间的矛盾。"

苏梦看了看题目，想了想："这道题应该选B，人民日益增长的美好生活需要和不平衡不充分的发展之间的矛盾。"

"对，"林默说道，"这是党的十九大报告中的重要内容。常识判断的时政部分需要关注国家大事，特别是党的代表大会、中央全会、政府工作报告等。花生十三总结了常识判断的高频考点，包括时政、法律、经济、历史、地理等方面的知识，掌握了这些考点，常识判断的正确率就能达到60%以上。"

接下来，林默给苏梦讲解了花生十三的常识判断积累方法，包括时政的关注重点、法律的常考法条、经济的基本概念、历史的重要事件、地理的常见考点等。他讲得非常详细，每一个考点都举了几个例子，苏梦听得很认真，不时点头表示理解。

"林默，你看这道题，"苏梦指着一道法律题说道，"题干是'根据《中华人民共和国刑法》，下列说法正确的是？选项是A.已满十四周岁不满十六周岁的人，犯故意杀人、故意伤害致人重伤或者死亡、强奸、抢劫、贩卖毒品、放火、爆炸、投毒罪的，应当负刑事责任 B.已满十六周岁的人犯罪，应当负刑事责任 C.已满十四周岁不满十八周岁的人犯罪，应当从轻或者减轻处罚 D.因不满十六周岁不予刑事处罚的，责令他的家长或者监护人加以管教；在必要的时候，也可以由政府收容教养。'问下列说法正确的是？"

苏梦看了看题目，想了想："根据刑法，已满十六周岁的人犯罪，应当负刑事责任，所以选项B正确。已满十四周岁不满十六周岁的人，犯故意杀人、故意伤害致人重伤或者死亡、强奸、抢劫、贩卖毒品、放火、爆炸、投毒罪的，应当负刑事责任，所以选项A正确。已满十四周岁不满十八周岁的人犯罪，应当从轻或者减轻处罚，所以选项C正确。因不满十六周岁不予刑事处罚的，责令他的家长或者监护人加以管教；在必要的时候，也可以由政府收容教养，所以选项D正确。所以正确答案是ABCD。"

"对，"林默说道，"这就是法律题的常考知识点，需要掌握刑法的基本法条。你做得很好。"

两人一直学到晚上六点，然后一起去食堂吃饭。吃饭时，苏梦问道："林默，你说常识判断的积累需要多长时间？"

"常识判断的积累是一个长期的过程，"林默说道，"需要每天关注时政新闻，每天积累一些知识点，长期坚持才能见效。"

"那我每天都关注时政新闻，每天积累一些知识点，"苏梦说道，"争取在国考前积累足够的常识。"

林默点了点头："好，这样很好。"

吃完饭，两人回到图书馆，继续学习。林默给苏梦讲解了花生十三的常识判断常考题型，包括时政、法律、经济、历史、地理等方面的知识。苏梦学得很认真，很快就掌握了这些知识点。

"林默，你看这道题，"苏梦指着一道历史题说道，"题干是'下列历史事件按时间先后顺序排列正确的是？①焚书坑儒 ②百家争鸣 ③独尊儒术 ④罢黜百家。选项是A.①②③④ B.②①④③ C.②①③④ D.①②④③。'"

苏梦看了看题目，想了想："百家争鸣是春秋战国时期，焚书坑儒是秦朝，罢黜百家、独尊儒术是汉朝，所以正确的顺序是②①④③。所以正确答案是B。"

"对，"林默说道，"这就是历史题的时间顺序，需要掌握重要历史事件的发生时间。你做得很好。"

两人一直学到晚上八点，然后一起走出图书馆。苏梦说道："林默，今天的学习收获很大，我觉得我的常识判断积累方法提高了很多。"

林默笑了笑："只要你坚持积累，你的常识判断水平会越来越高，正确率会越来越高。"

"谢谢你，林默，"苏梦说道，"如果没有你，我可能永远都不会掌握这些方法。"

"不用谢，"林默说道，"我们是朋友，应该互相帮助。"

苏梦笑着点了点头，转身离开了。林默看着她的背影，心中涌起一股暖流。他知道，这一次，他一定能和苏梦在一起，创造属于自己的幸福生活。

回到宿舍，林默躺在床上，回想着今天和苏梦的相处，觉得很开心。他拿出笔记本，开始总结今天的学习内容，然后制定明天的学习计划。

林默看着自己的未来规划，心中充满了希望。他知道，有了重生的机会和花生十三的技巧，他一定能实现自己的目标，创造属于自己的传奇。
