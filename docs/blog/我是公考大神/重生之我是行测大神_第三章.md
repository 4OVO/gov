# 第三章 规划未来

林默醒来时，天已经大亮了。他看了看手机，已经八点了。他迅速起床洗漱，然后去食堂吃早餐。

今天是周六，图书馆人很多，林默和苏梦找了很久才找到一个空位。

"今天我们学判断推理吧，"林默说道，"判断推理包括图形推理、定义判断、类比推理和逻辑判断四个部分，也是行测的重要组成部分。"

苏梦点了点头："好，我图形推理总是做不好，经常找不到规律。"

林默翻开教材，找到一道图形推理题："你看这道题，题干中的图形都是由两个部分组成的，而且这两个部分都是对称图形。第一个图形是两个半圆，第二个是两个三角形，第三个是两个矩形，第四个应该是什么？"

苏梦看了看题目，想了想："应该是两个对称图形组成的，选项中只有C选项是两个对称图形。"

"对，"林默说道，"这就是图形推理的对称性规律。花生十三总结了图形推理的常见规律，包括对称性、数量关系、位置变化、样式运算等，掌握了这些规律，图形推理就变得简单了。"

接下来，林默给苏梦讲解了花生十三的判断推理技巧，包括图形推理的规律总结、定义判断的关键词法、类比推理的逻辑关系、逻辑判断的翻译推理等。苏梦学得很认真，很快就掌握了这些技巧。

"林默，你讲得真好，"苏梦说道，"比我们学校的老师讲得还清楚。"

林默笑了笑："谢谢夸奖，我只是把花生十三的技巧讲给你听而已。"

两人一直学到中午，然后一起去食堂吃饭。吃饭时，苏梦问道："林默，你有详细的备考计划吗？"

"有啊，"林默说道，"我打算从现在开始，每天学习行测和申论各四个小时，直到国考结束。"

"国考是什么时候？"苏梦问道。

"2019年国考应该在12月举行，"林默说道，"还有半年的时间，足够我们备考了。"

"半年时间？"苏梦惊讶地说道，"那不是只有六个月了吗？时间会不会太紧？"

"不会，"林默说道，"六个月的时间足够了，只要我们合理安排时间，掌握正确的学习方法，一定能考上公务员。"

"那你能给我看看你的备考计划吗？"苏梦问道。

林默从书包里拿出笔记本，翻开其中一页，上面写着详细的备考计划：

**备考计划（2018年6月-2019年12月）**

**第一阶段（6月-8月）：基础学习**
- 资料分析：每天2小时，掌握速算技巧和常考公式
- 言语理解：每天2小时，掌握解题技巧和成语积累
- 判断推理：每天2小时，掌握各种规律和逻辑关系
- 数量关系：每天1小时，掌握基础题型和秒杀技巧
- 常识判断：每天0.5小时，积累时政和基础知识
- 申论：每天2小时，掌握题型和写作技巧

**第二阶段（9月-10月）：强化训练**
- 模块刷题：每天4小时，每个模块刷100道题
- 套题训练：每周2套行测真题，1套申论真题
- 错题整理：每天1小时，整理错题并分析原因
- 申论积累：每天1小时，积累热点和素材

**第三阶段（11月-12月）：冲刺阶段**
- 真题模拟：每天1套行测真题，1套申论真题
- 限时训练：严格按照考试时间做题，提高速度
- 查漏补缺：针对薄弱环节进行专项训练
- 心态调整：保持良好的心态，积极备考

苏梦看了看备考计划，惊讶地说道："你的计划真详细，我都不知道国考还要考申论。"

"申论是公务员考试的重要组成部分，"林默说道，"行测和申论各占100分，申论的分数也很重要。"

"那你申论学得怎么样？"苏梦问道。

"还可以，"林默说道，"申论主要靠积累和练习，只要掌握了题型和写作技巧，得分并不难。"

"那你能教我申论吗？"苏梦问道。

"当然可以，"林默说道，"从下周开始，我们每天下午学申论。"

苏梦点了点头："好，那我就按照你的计划来备考。"

吃完饭，两人回到图书馆，继续学习。林默给苏梦讲解了花生十三的数量关系技巧，包括行程问题、工程问题、利润问题、几何问题等常见题型的秒杀公式。苏梦学得很认真，很快就掌握了这些技巧。

"林默，你说我们能考上公务员吗？"苏梦突然问道。

"能，"林默坚定地说道，"只要我们按照计划认真备考，一定能考上公务员。"

苏梦看着林默坚定的眼神，心中涌起一股信心："嗯，我相信你。"

下午四点，林默和苏梦结束了学习。他们一起走出图书馆，苏梦问道："林默，你毕业后打算做什么？"

"我打算先考上公务员，"林默说道，"然后可能会做一些公考培训的工作，帮助更多的人考上公务员。"

"公考培训？"苏梦问道，"你想当老师吗？"

"嗯，"林默说道，"我觉得花生十三的技巧很有用，应该让更多的人知道这些技巧，帮助他们考上公务员。"

"那如果我们都考上了公务员，你还会做公考培训吗？"苏梦问道。

"会啊，"林默说道，"我可以利用业余时间做，既不影响工作，又能帮助别人。"

苏梦点了点头："这是个不错的想法。"

两人走到校门口，苏梦说道："我要回宿舍了，明天见。"

"明天见，"林默说道，"记得按照计划学习哦。"

苏梦笑着点了点头，转身离开了。林默看着她的背影，心中涌起一股暖流。他知道，这一次，他不仅要考上公务员，还要和苏梦在一起，创造属于他们的幸福生活。

回到宿舍，林默躺在床上，思考着自己的未来。他重生到了2018年，不仅获得了花生十三的行测技巧，还保留了前世的记忆。他可以利用这些优势，在公考中取得成功，同时在其他方面也有所作为。

他想到，2018年是短视频开始兴起的年份，抖音、快手等平台正在快速发展。如果他现在开始做公考培训的短视频，可能会比花生十三更早走红，成为公考培训行业的佼佼者。

不过，他现在的首要目标是考上公务员，改变自己的命运。至于做短视频，那是考上公务员之后的事情。

林默拿出笔记本，在备考计划的基础上，又增加了一些内容：

**未来规划**
1. 2018年6月-12月：认真备考，参加2019年国考
2. 2019年1月-3月：准备面试，考上公务员
3. 2019年4月-12月：入职公务员，熟悉工作
4. 2020年1月-12月：开始做公考培训的短视频，分享行测技巧
5. 2021年及以后：扩大影响力，成为公考培训行业的佼佼者

林默看着自己的未来规划，心中充满了希望。他知道，有了重生的机会和花生十三的技巧，他一定能实现自己的目标，创造属于自己的传奇。
