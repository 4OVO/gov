# 第十二章 技巧总结

周一上午，林默和苏梦一起去图书馆学习。他们已经参加了模考，检验了自己的学习成果，现在是时候总结一下行测和申论的技巧，以便在国考前进一步提高成绩。

"林默，我们已经学习了一个多月，参加了一次模考，现在应该总结一下行测和申论的技巧，"苏梦说道。

"好啊，"林默说道，"我已经总结了一些行测和申论的技巧，我们可以一起讨论。"

林默拿出笔记本，开始给苏梦讲解他总结的行测技巧：

**行测技巧总结**

**一、资料分析**
1. **速算技巧**：截位直除法、估算法、特殊分数法、化同法、差分法
2. **常考公式**：基期量、现期量、增长率、增长量、比重、平均数、倍数
3. **时间分配**：20道题，25分钟
4. **正确率目标**：90%以上

**二、言语理解**
1. **逻辑填空**：对应关系、词语辨析、语境分析
2. **片段阅读**：结构分析、主题词定位、选项分析
3. **语句排序**：首尾句判断、关联词捆绑、时间顺序
4. **时间分配**：40道题，35分钟
5. **正确率目标**：80%以上

**三、判断推理**
1. **图形推理**：对称性、数量关系、位置变化、样式运算
2. **定义判断**：关键词法、排除法
3. **类比推理**：逻辑关系、词性分析
4. **逻辑判断**：翻译推理、加强削弱、真假推理
5. **时间分配**：40道题，35分钟
6. **正确率目标**：80%以上

**四、数量关系**
1. **常考题型**：行程问题、工程问题、利润问题、几何问题、排列组合问题
2. **秒杀技巧**：速度比、效率比、成本计算、面积公式
3. **时间分配**：15道题，15分钟
4. **正确率目标**：60%以上

**五、常识判断**
1. **积累方法**：每天关注时政新闻，每天积累知识点
2. **常考知识点**：时政、法律、经济、历史、地理
3. **时间分配**：20道题，10分钟
4. **正确率目标**：60%以上

**六、行测整体技巧**
1. **答题顺序**：资料分析→言语理解→判断推理→数量关系→常识判断
2. **时间分配**：120分钟，135道题
3. **涂卡技巧**：做完一个模块涂一次卡
4. **心态调整**：保持冷静，遇到难题跳过

苏梦认真听着林默的讲解，不时点头表示理解。她觉得林默总结的技巧很详细，很有用。

"林默，你总结的行测技巧很详细，很有用，"苏梦说道，"那申论技巧呢？"

"申论技巧我也总结了，"林默说道，"我们接下来就讲申论技巧。"

林默拿出笔记本，开始给苏梦讲解他总结的申论技巧：

**申论技巧总结**

**一、归纳概括题**
1. **解题技巧**：提取关键词，分类概括
2. **注意事项**：紧扣材料，不要主观臆断
3. **字数要求**：150-200字
4. **分值**：10-15分

**二、综合分析题**
1. **解题技巧**：辩证思维，分析原因和影响
2. **注意事项**：观点明确，逻辑清晰
3. **字数要求**：200-300字
4. **分值**：15-20分

**三、提出对策题**
1. **解题技巧**：针对问题，提出具体可行的对策
2. **注意事项**：对策要具体，要有可操作性
3. **字数要求**：200-300字
4. **分值**：15-20分

**四、贯彻执行题**
1. **解题技巧**：掌握不同文种的格式要求
2. **注意事项**：格式正确，内容符合要求
3. **字数要求**：300-500字
4. **分值**：20-25分

**五、申发论述题**
1. **解题技巧**：结构安排，论点明确，论据充分
2. **注意事项**：结合材料，联系实际
3. **字数要求**：1000-1200字
4. **分值**：35-40分

**六、申论整体技巧**
1. **时间分配**：180分钟，5道题
2. **写作技巧**：字迹工整，逻辑清晰
3. **积累方法**：每天看人民日报，每天写一篇申论
4. **心态调整**：保持冷静，遇到难题跳过

苏梦认真听着林默的讲解，不时点头表示理解。她觉得林默总结的申论技巧很详细，很有用。

"林默，你总结的申论技巧很详细，很有用，"苏梦说道，"我一定会按照这些技巧来学习申论。"

"好，"林默说道，"只要你按照这些技巧来学习，你的申论成绩一定会提高。"

两人一直学到中午，然后一起去食堂吃饭。吃饭时，苏梦问道："林默，你说我们在国考前还需要做什么？"

"我们在国考前还需要做以下几件事，"林默说道，"第一，每天做一套行测和申论真题，保持手感；第二，每天总结错题，分析错误原因；第三，每天积累常识和申论素材；第四，调整心态，保持良好的状态。"

"好，"苏梦说道，"我们一定按照这些要求来做。"

吃完饭，两人回到图书馆，继续学习。林默给苏梦讲解了他总结的行测和申论技巧的具体应用，苏梦听得很认真，很快就掌握了这些技巧。

"林默，你说我们什么时候才能收到国考的准考证？"苏梦问道。

"国考的准考证一般在考试前一周左右可以打印，"林默说道，"2019年国考应该在11月底或12月初打印准考证。"

"那我们还有两个月的时间备考，"苏梦说道，"我们要继续努力，争取在国考前进一步提高成绩。"

"对，"林默说道，"我们要继续努力，争取在国考中取得好成绩。"

两人一直学到晚上八点，然后一起走出图书馆。苏梦说道："林默，今天的学习收获很大，我觉得我对行测和申论的技巧有了更全面的了解。"

"我也觉得今天的学习收获很大，"林默说道，"我们总结了行测和申论的技巧，为国考做好了准备。"

"谢谢你，林默，"苏梦说道，"如果没有你，我可能永远都不会掌握这些技巧。"

"不用谢，"林默说道，"我们是朋友，应该互相帮助。"

苏梦笑着点了点头，转身离开了。林默看着她的背影，心中涌起一股暖流。他知道，这一次，他一定能和苏梦一起考上公务员，创造属于自己的幸福生活。

回到宿舍，林默躺在床上，回想着今天和苏梦的相处，觉得很开心。他已经总结了行测和申论的技巧，为国考做好了准备。他相信，只要他和苏梦继续努力，一定能在国考中取得好成绩，考上公务员。

林默看着自己的未来规划，心中充满了希望。他知道，有了重生的机会和花生十三的技巧，他一定能实现自己的目标，创造属于自己的传奇。
