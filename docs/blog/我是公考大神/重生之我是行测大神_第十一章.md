# 第十一章 模考初体验

周日上午，林默和苏梦一起去图书馆学习。他们已经学习了一个多月，掌握了花生十三的行测和申论技巧，现在是时候检验一下自己的学习成果了。

"林默，我们学习了一个多月，应该参加一次模考，检验一下自己的学习成果，"苏梦说道。

"好啊，"林默说道，"我知道中公教育每周日都会组织免费的模考，我们可以去参加。"

"那太好了，"苏梦说道，"我们现在就去中公教育参加模考吧。"

林默点了点头，和苏梦一起去中公教育参加模考。他们来到中公教育的教室，发现已经有很多人在那里了。他们找了两个空位坐下，等待模考开始。

模考开始后，林默和苏梦拿到了行测试卷。林默快速浏览了一下试卷，发现试卷的题型和难度与国考相似。他按照花生十三的答题顺序，先做资料分析，然后做言语理解，接着做判断推理，最后做数量关系和常识判断。

林默做资料分析时，使用了花生十三的截位直除法、估算法、特殊分数法等速算技巧，很快就做完了资料分析部分，正确率很高。然后他做言语理解，使用了花生十三的对应关系、结构分析等技巧，也很快就做完了言语理解部分。接着他做判断推理，使用了花生十三的规律总结、关键词法等技巧，同样很快就做完了判断推理部分。最后他做数量关系和常识判断，使用了花生十三的秒杀技巧和积累的知识点，也很快就做完了。

苏梦也按照林默教她的答题顺序和技巧做试卷，她发现自己的做题速度比以前快了很多，正确率也提高了很多。

模考结束后，林默和苏梦交了试卷，等待成绩公布。成绩公布后，林默看到自己的行测成绩是85分，苏梦的行测成绩是75分，两人都很开心。

"林默，你的行测成绩是85分，好厉害啊，"苏梦说道。

"你的行测成绩是75分，也很厉害啊，"林默说道，"比你以前的成绩提高了很多。"

"这都是你的功劳，"苏梦说道，"如果没有你教我花生十三的技巧，我不可能考这么高的分数。"

"不用谢，"林默说道，"这是你自己努力的结果。"

这时，中公教育的王老师走了过来，对林默说道："林默同学，你的行测成绩是85分，在我们这里的模考中排名第一，恭喜你。"

"谢谢王老师，"林默说道。

"林默同学，我们中公教育想邀请你参加我们的公考培训课程，担任助教，"王老师说道，"待遇很优厚，你考虑一下吧。"

"谢谢王老师的邀请，"林默说道，"但是我现在正在备考公务员，没有时间担任助教。"

"没关系，"王老师说道，"我们可以等你考上公务员之后再邀请你。"

林默点了点头："好的，谢谢王老师。"

离开中公教育后，林默和苏梦一起去食堂吃饭。吃饭时，苏梦问道："林默，你的行测成绩是85分，已经很高了，你觉得你能考上公务员吗？"

"能，"林默说道，"我的行测成绩已经达到了国考的要求，只要申论成绩也能达到70分以上，我就能考上公务员。"

"那你的申论成绩能达到70分以上吗？"苏梦问道。

"能，"林默说道，"我每天都写申论，掌握了花生十三的申论技巧，申论成绩应该能达到70分以上。"

"那太好了，"苏梦说道，"我相信你一定能考上公务员。"

"你也能考上公务员，"林默说道，"你的行测成绩是75分，申论成绩只要能达到65分以上，就能考上公务员。"

"真的吗？"苏梦问道，"我申论成绩能达到65分以上吗？"

"能，"林默说道，"你每天都写申论，掌握了花生十三的申论技巧，申论成绩应该能达到65分以上。"

"那太好了，"苏梦说道，"我相信我也能考上公务员。"

吃完饭，林默和苏梦回到图书馆，继续学习。林默给苏梦讲解了模考中出现的错题，帮助她分析错误原因，提高解题能力。苏梦听得很认真，很快就掌握了这些知识点。

"林默，你说我们什么时候才能参加国考？"苏梦问道。

"2019年国考应该在12月举行，"林默说道，"还有四个月的时间，足够我们备考了。"

"那我们要继续努力，"苏梦说道，"争取在国考前把行测和申论的成绩再提高一些。"

"对，"林默说道，"我们要继续努力，争取在国考中取得好成绩。"

两人一直学到晚上八点，然后一起走出图书馆。苏梦说道："林默，今天的模考让我看到了自己的进步，我觉得我一定能考上公务员。"

"我也觉得你一定能考上公务员，"林默说道，"只要你继续努力，一定能考上公务员。"

"谢谢你，林默，"苏梦说道，"如果没有你，我可能永远都不会有这样的进步。"

"不用谢，"林默说道，"我们是朋友，应该互相帮助。"

苏梦笑着点了点头，转身离开了。林默看着她的背影，心中涌起一股暖流。他知道，这一次，他一定能和苏梦一起考上公务员，创造属于自己的幸福生活。

回到宿舍，林默躺在床上，回想着今天的模考，觉得很开心。他的行测成绩是85分，已经达到了国考的要求，申论成绩只要能达到70分以上，就能考上公务员。他相信，只要他继续努力，一定能考上公务员，实现自己的目标。

林默看着自己的未来规划，心中充满了希望。他知道，有了重生的机会和花生十三的技巧，他一定能实现自己的目标，创造属于自己的传奇。
